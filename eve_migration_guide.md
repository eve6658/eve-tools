## 三、新电脑：安装部署（5分钟）

### 1. 安装 Node.js

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo bash -
sudo apt install -y nodejs
node -v  # 应输出 v20.x.x
```

### 2. 安装 OpenClaw

```bash
npm install -g openclaw
openclaw --version
```

### 3. 初始化 OpenClaw

```bash
openclaw init
```

### 4. 解压迁移包到工作区

```bash
cd ~/.openclaw/workspace
tar xzf ~/Desktop/eve_migration.tar.gz
```

### 5. 安装 Python 依赖

```bash
pip3 install python-docx --break-system-packages
```

### 6. 配置 QQ Bot

```bash
# 替换为你的 AppID 和 AppSecret
openclaw channels add --channel qqbot --token "你的AppID:你的AppSecret"
```

### 7. 配置 Tushare（可选，需要股票数据时）

```bash
echo 'export TUSHARE_TOKEN=你的TushareToken' >> ~/.bashrc
source ~/.bashrc
```

### 8. 启动 OpenClaw

```bash
openclaw gateway start
```

### 9. 验证安装

```bash
# 测试 Eve Evolution 框架
cd ~/.openclaw/workspace
python3 -m eve_evolution.demo

# 测试 Buddy 系统
python3 -c "from eve_evolution.buddy import *; c = generate_companion('test'); print(c.full_display())"

# 测试策略引擎
python3 -c "from eve_evolution.strategy import *; s = quick_parse('打板策略'); print(f'策略: {s.name}')"
```

## 六、常见问题

**Q: 新电脑上没有 QQ Bot Token 怎么办？**
A: 登录 QQ 开放平台（https://q.qq.com），在你的应用里查看 AppID + AppSecret

**Q: Tushare 数据获取失败？**
A: 检查 token：`echo $TUSHARE_TOKEN`，没有就去 Tushare Pro 注册

**Q: python-docx 导入失败？**
A: `pip3 install python-docx --break-system-packages`（Linux 需要 --break-system-packages）

**Q: eve_evolution/demo.py 跑不动？**
A: 检查 Python 版本：`python3 --version`，需要 >= 3.10
