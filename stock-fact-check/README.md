# 事实调查分析框架

## 数据源

| 来源 | 用途 | 类型 |
|------|------|------|
| 慧博投研 (hibor.com.cn) | 研报、行业分析、分析师观点 | 定性 |
| 东方财富 (eastmoney.com) | 公告、财务摘要、行情 | 半结构化 |
| AkShare (Python) | 免费A股行情/财务数据 | 结构化 |
| 天气/搜索 | 宏观背景信息 | 辅助 |

## 调查模板

每个标的的分析报告包含以下部分：

### 1. 公司画像
- 公司名称 / 代码 / 所属行业
- 主营业务一句话
- 核心产品/服务

### 2. 财务体检（最近3-5年趋势）
- 营收增速趋势
- 净利润增速趋势
- 毛利率 / 净利率
- ROE（净资产收益率）
- 资产负债率
- 经营性现金流

### 3. 估值判断
- 当前PE（TTM）vs 行业均值
- 当前PB vs 行业均值
- 历史PE分位数

### 4. 资金面
- 近期主力资金流向
- 北向资金持仓变化
- 股东人数变化

### 5. 研报观点汇总
- 最近3-5篇慧博研报核心观点
- 券商评级汇总
- 分析师共识目标价（如有）

### 6. 风险提示
- 主要风险因素
- 需要持续跟踪的指标

### 7. 综合判断
- 评分（1-10）
- 核心逻辑一句话
- 建议动作（关注/等待/买入/卖出）
