## 一、核心认知升级：量化改变了什么

### 1.1 传统理论的失效

传统盘口理论建立在一个前提上：**主力和散户是博弈双方**。

2024年以后，量化交易占A股日均成交额比例超过30%，博弈格局变成三方：

```
传统：主力 vs 散户
现在：主力 vs 量化 vs 散户
```

量化的特点是**机械执行**：跌破某个价位自动止损，突破某个价位自动追涨。
这给了主力一个新的武器——**利用量化的机械性制造假信号**。

### 1.2 三方面博弈的新规则

| 参与者 | 行为特征 | 弱点 |
|--------|----------|------|
| 散户 | 情绪驱动，追涨杀跌 | 恐慌时交筹码 |
| 量化 | 程序驱动，固定策略 | 机械性可被利用 |
| 主力 | 意图驱动，灵活应变 | 资金有限 |

**主力的新策略：先触发量化的机械止损，再利用散户的恐慌，收集低价筹码后快速拉升，让量化追涨时已经来不及。**

## 二、反向运行理论

### 2.1 传统理论："主力向阻力最小的方向运行"

传统认知：主力会选择阻力最小的方向——下方没支撑就向下，上方没压力就向上。

### 2.2 量化时代升级版："主力先反向运行，再向真正的阻力最小方向运行"

**核心修正：阻力最小方向是"最终方向"，不是"初始方向"。**

主力的操作分为两步：

```
第一步：反向运行 → 向"阻力大"的方向砸 → 触发量化止损 + 散户恐慌
第二步：正向运行 → 向"阻力真正小"的方向拉 → 筹码已收够，拉升不费力
```

### 2.3 实战案例：002788 鹭燕（2026/4/3）

**背景：** 4/2尾盘抢筹+5.97%，市场预期4/3继续涨。

| 时间 | 价格 | 主力行为 | 量化/散户反应 |
|------|------|----------|---------------|
| 4/2 14:54 | 16.97 | 连续B大单抢筹 | 量化程序锁定"强势股" |
| **4/3 09:26** | **16.49** | **低开-3.23%** | 量化触发止损，散户恐慌 |
| 09:26-09:37 | 16.46-16.48 | 被动小单接盘 | 散户继续抛，量化继续卖 |
| 09:37 | 16.46 | 275笔买盘+390手大单 | 筹码收集接近完成 |
| 09:41 | 16.48 | 477手主动B单 | **转为主动进攻** |
| 09:49 | 17.36 | 快速拉升 | 量化追涨，但筹码已被主力拿走 |
| **10:44** | **18.38** | **+7.9%** | 主力完成一轮反向收割 |

**关键数据：**
- 低开幅度：-3.23%（触发量化止损阈值）
- 买盘变化：14笔→275笔（主力从伪装到暴露）
- 卖盘特征：平均23手/笔（散户碎片化抛售）
- 买盘特征：从10手/笔→390手/笔→477手（主力从被动到主动）
- 最终涨幅：从-3.23%到+7.9%，**振幅11.1%**

## 三、缩量：唯一做不了假的信号

### 3.1 为什么缩量无法伪造？

| 信号 | 能否做假 | 原因 |
|------|----------|------|
| 放量 | ✅ 能 | 主力对倒（自己买自己卖）可制造虚假成交量 |
| K线形态 | ✅ 能 | 尾盘拉升画线、集合竞价做收盘价 |
| 盘口挂单 | ✅ 能 | 大单秒撤，挂了不等于成交 |
| 均线 | ✅ 能 | 拉一下价格就能改变均线形态 |
| **缩量** | **❌ 不能** | 缩量=市场参与者少，主力无法通过"不交易"制造假象 |

**缩量的本质：**
- 缩量上涨 = 筹码高度锁定，主力不用花大力气就能拉
- 缩量下跌 = 主力没卖，是散户和量化在恐慌
- 缩量横盘 = 主力在等，没有出货意图

### 3.2 缩量验证规则

**规则一：大跌缩量 = 洗盘，不是出货**

| 案例 | 大跌日 | 跌幅 | 成交量 | 次日走势 |
|------|--------|------|--------|----------|
| 600666 | 3/31 | -8.65% | 637万手 | **4/1涨停+10%** |
| 000890 | 3/26 | -9.99% | 缩量 | **3/27涨停+10.04%** |
| 002730 | 3/26 | -10.02% | 缩量 | **3/27反弹+4.94%** |

**对比：放量下跌 = 出货**

| 案例 | 大跌日 | 跌幅 | 成交量 | 后续走势 |
|------|--------|------|--------|----------|
| 002310 | 3/26 | -8.2% | 放量 | 3/27反弹后继续跌，4/2跌到3.03 |

**判断标准：大跌日成交量 < 前5日平均成交量 = 缩量下跌 = 洗盘**

**规则二：涨停缩量 = 高度控盘**

600666 4/1涨停：
- 涨幅：+10.00%
- 成交量：264万手
- 前一日（3/31）成交量：637万手
- **量比：0.41倍**

涨停缩量说明：
1. 散户手里筹码少（被主力收集了）
2. 主力不需要花大力气封板
3. 后续还有上涨空间

**规则三：低开缩量 = 不是主力在砸**

002788 4/3 09:26：
- 低开-3.23%
- 买盘14笔，平均10手/笔
- 卖盘2笔，平均23手/笔

低开缩量说明：
1. 主力没有主动卖出
2. 低开是量化止损盘和散户恐慌造成的
3. 主力在下方被动接盘

### 3.3 缩量是所有盘口分析的"验证器"

| 分析方法 | 信号 | 配合缩量 |
|----------|------|----------|
| 小单占比 | 卖盘小单>80% = 吸筹 | +缩量 = 确认主力没跑 |
| 铁底支撑 | 5.70有26,500手 | +缩量 = 铁底可信，砸不动 |
| 反向运行 | 破支撑后快速拉回 | +缩量 = 确认是洗盘 |
| 盘口质量 | 买盘大单/卖盘小单 | +缩量 = 主力控盘度极高 |

**口诀：缩量不说谎。其他信号告诉你主力在做什么，缩量告诉你主力说的是真话。**

## 五、旁观者视角框架

### 5.1 核心问题

> **"如果我现在空仓，看到这个盘口，会不会买入？"**

### 5.2 使用场景

- 持仓中犹豫要不要卖的时候
- 浮盈缩水感到恐惧的时候
- 被盘面波动影响判断的时候

### 5.3 逻辑检验

| 你的回答 | 意味着 | 操作 |
|----------|--------|------|
| 会买入 | 持有=买入，方向一致 | 继续持有 |
| 不会买入 | 持有≠买入，逻辑矛盾 | 考虑卖出 |
| 不确定 | 信号不明确 | 严格止损，观望 |

### 5.4 600666 4/3实操验证

**假设你空仓，看到以下盘口：**

| 信号 | 空仓者视角 |
|------|-----------|
| 买1 13,300手/2笔（09:27） | "主力明牌做多" |
| 卖1 15笔/37手/笔 | "散户碎片化抛售" |
| 5.70有26,500手终极防线 | "主力死守，跌不动" |
| 下跌缩量（545万 vs 昨日746万） | "主力没卖" |

**空仓者判断：会在5.70-5.80建仓。**

**结论：既然空仓都愿意买，持有更不应该卖。**

### 5.5 与持仓偏见的对抗

| 持仓偏见 | 旁观者视角 |
|----------|-----------|
| "浮盈从19%缩水到5%，该跑了" | "铁底不破+缩量，主力还在" |
| "跌了这么多，是不是主力跑了" | "缩量=主力没跑，是散户在跑" |
| "要不要先卖了等跌了再买回来" | "你能确定跌吗？铁底26,500手" |

**核心：持仓偏见放大恐惧，旁观者视角恢复理性。**

## 六、暴力洗盘案例库

### 6.1 定义

暴力洗盘 = 主力在上升趋势中，用**单日大跌/跌停**清洗浮筹，次日快速拉回。

### 6.2 判断标准

| 条件 | 说明 |
|------|------|
| 前置趋势 | 处于上升趋势中 |
| 洗盘日 | 大跌>7%（最好跌停） |
| 量能 | **缩量**（关键！） |
| 次日 | 快速反弹>3% |
| 后续 | 3日内收复洗盘日跌幅 |

### 6.3 案例一：000890 法尔胜

| 日期 | 收盘 | 涨跌幅 | 量 | 解读 |
|------|------|--------|-----|------|
| 3/25 | 13.61 | +6.41% | 放量 | 拉升制造关注 |
| **3/26** | **12.25** | **-9.99%** | **缩量** | ⬇️ 跌停洗盘 |
| **3/27** | **13.48** | **+10.04%** | 放量 | ⬆️ 次日涨停拉回 |
| 3/30 | 14.83 | +10.01% | 放量 | 连续涨停 |
| 3/31 | 15.43 | +4.05% | — | 继续上攻 |
| 4/3 | 16.01 | — | — | 从洗盘价12.25涨到16.01，**+30.7%** |

**洗盘特征：**
- 跌停日缩量 → 主力没卖
- 次日涨停 → 主力快速拉回，不给低位上车机会
- 后续持续走强 → 确认洗盘

### 6.4 案例二：002730 电科芯片

| 日期 | 收盘 | 涨跌幅 | 解读 |
|------|------|--------|------|
| 3/25 | 25.86 | +10.00% | 涨停 |
| **3/26** | **23.27** | **-10.02%** | ⬇️ 跌停洗盘 |
| **3/27** | **24.42** | **+4.94%** | ⬆️ 反弹 |
| 3/31 | 26.50 | +8.56% | 收复 |
| 4/1 | 29.15 | +10.00% | 涨停 |
| 4/2 | 29.49 | +1.17% | 从洗盘价23.27涨到29.49，**+26.7%** |

### 6.5 案例三：600666 奥瑞德（本轮持仓标的）

- 后续继续跌 → 确认是假反弹

## 七、综合判断流程（操作检查清单）

### 7.1 盘中判断流程

```
第一步：看量能
  ├─ 下跌缩量？ → 主力没跑，可能是洗盘
  ├─ 下跌放量？ → 主力可能在出货，警惕
  └─ 上涨缩量？ → 高度控盘，持有
  └─ 上涨放量？ → 正常，看其他指标

第二步：看盘口
  ├─ 买盘大单 + 卖盘小单 → 吸筹
  ├─ 买盘小单 + 卖盘大单 → 出货
  └─ 双方小单 → 散户主导，观望

第三步：看铁底
  ├─ 下方有大单托底 → 主力防守，可信
  ├─ 下方无支撑 → 可能继续跌
  └─ 铁底被击穿但缩量 → 可能是反向洗盘

第四步：旁观者视角
  └─ "如果空仓，我会买吗？"
      ├─ 会 → 持有
      └─ 不会 → 考虑卖出
```

### 7.2 关键阈值速查

| 指标 | 吸筹信号 | 出货信号 | 洗盘信号 |
|------|----------|----------|----------|
| 卖盘小单占比 | >80% | <30% | 高但缩量 |
| 买盘小单占比 | <30% | >80% | — |
| 大跌日量能 | — | — | <前5日均量 |
| 次日走势 | — | — | 反弹>3% |
| 铁底 | 有且不破 | 无 | 破后快速收回 |

### 8.3 今日总结的法则

1. **缩量不说谎** — 大跌缩量=洗盘，涨停缩量=控盘
2. **反向运行** — 主力先向阻力大的方向砸，再向阻力小的方向拉
3. **旁观者视角** — "空仓会买吗？" 消除持仓偏见
4. **被动→主动** — 盘口从被动接单演变到主动吃筹，是启动信号
5. **铁底验证** — 有铁底的急跌是洗盘，无铁底的急跌是出货
