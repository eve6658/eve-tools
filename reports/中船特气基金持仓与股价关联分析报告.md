## 一、核心结论

**基金资金流是科创板股票价格的首要驱动力，而非企业基本面。实时追踪基金持仓变化，是把握买卖时机的关键。**

中船特气近一个月股价上涨80%（41→74元），同期企业经营无重大变化。驱动因素完全来自基金申购带来的被动买入资金流。谁掌握基金持仓数据，谁就掌握了价格方向的"油门"。

### 2.3 为什么ETF增持推动了股价上涨？

```
机制链条：
  新发半导体基金 → 散户/机构申购 → 基金募集到资金
  → 基金必须按指数权重买入成分股
  → 中船特气是科创板半导体材料成分股 → 被动买入
  → 大量被动买盘推高价格

Q1数据（3月31日截止）：ETF已开始增持
4月行情（41→74）：ETF加速增持（Q2报告才会体现）
→ 基金资金流时间线 ≈ 股价上涨时间线
```

## 六、附录：分析框架概要

本报告基于的"主力行为分析框架"核心观点：

1. **基金申购赎回是"油门"**，决定股价方向
2. **企业财报是"涡轮增压"**，决定涨跌速度
3. 有钱进来 + 财报好 → 涨得猛；有钱进来 + 财报普通 → 也涨
4. 赎回窗口是低位买入机会，募资窗口是持有/加仓信号
5. 实时基金持仓数据是追踪资金流的核心工具
