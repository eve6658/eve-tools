# 冬生其力 — 人性的本质

> 来源：抖音直播连麦，路飞（主播）与冬生（3000万资产股民）的对话
> 整理：Eve，2026-05-01

---

## 一、冬生的背景

冬生，1970年生，上海人，知青后代。从小跟外公外婆长大，性格内向，不会说话但会做事。在七个单位都因打架离职——不是主动惹事，是一直被欺负。

35岁发现自己不适合上班，开始靠股票改变命运。从最落魄时买不起米、喝自来水，到如今资产约3000万。

两段婚姻，均以失败告终。

---

## 二、两段婚姻的轨迹

### 第一段婚姻

前妻是同事，主动追的他。冬生不喜欢她，但当时太穷，有人愿意跟他就结了。

结婚条件：不办酒席、不领结婚证、不要小孩。

他说："我不喜欢你，只是为结婚而结婚。但我承诺过，哪天赚到钱了，我要找自己心爱的人。你的这一辈子遇到自己喜欢的人了，你去找。"

七年婚姻，恋爱同居一年，结婚六年。最后他走了，前妻如约找到了自己的人。

他对她的感情，像对女儿、对妹妹——不是爱情。

### 第二段感情

2020年9月，在浙江一个小城，冬生认识了第二个女人。她50岁，在加油站上班，坐一休一，工资六七千。

冬生的做法：拿着身份证和股票账户去找她。在她家楼下等了一个半月，不敢下车。最后鼓起勇气表白，要了电话号码。

在一起后，冬生承诺每月给她一万块生活费。后来谈到结婚，女方要求领证，冬生坚持要做财产公证。女方说："我们山里没有财产公证这回事，你要公证就是不信我。"

冬生妥协了。给她买了168万的房子（全款，没还价），130万装修。2024年8月19日，把房子100%过户给她。

四个月后，2024年12月22日，女方走了。理由是"你外面有女人"。

冬生没有女人。他每天在家看书看股票。

## 三、路飞的诊断：人性的本质

### 1. 你是一个极度缺爱的人

冬生的缺爱，不是表面能看出来的。他不会表达，但他每一次付出，都是在索取反馈。

他借钱给哥哥，2200美金，28年后才要回来。他把存款给哥哥炒股，反复被骗。他给女方房子、给生活费——**每一次付出，背后都有一个隐含的期待：你必须给我对等的回应。**

但人性是反道德的。你借钱给别人，道德上你应该是好人。但对方拿到钱后，他的反应不是感恩，而是觉得你欠他的。

**你永远在考虑的不是钱本身的价值，而是能不能得到反馈。但借钱和得到反馈，是两码事。**

### 2. 你用钱代替了自己的价值

冬生第一次见女方就亮股票账户。

路飞问："你为什么让他看你的钱？"

冬生："差不多是我的主要价值。"

路飞："因为你害怕你本人没有办法有足够的吸引力来吸引她。"

**钱的本质是人与人之间的不信任。** 当你用钱作为唯一的价值展示时，你传递的信息是："我这个人本身不值得被爱，只有钱值得。"

### 3. 因果不能强行嫁接

春天种下西瓜种子，秋天能不能结出西瓜，中间需要很多过程。有随机性，有各种变量。

这些变量，路飞称之为"天选"——她儿子怎么选择、她父母怎么选择、她前夫怎么选择，决定了你们两个人之间这个"瓜"能不能成。

**你试图用钱强行加速因果：我给你钱→你应该爱我。但因果之间不是直挂的关系。**

你拿出3000万的股票账户给别人看，就是花最短的时间强行嫁接因果。代价很大。你吃亏吃这么久，就是因为这个原因。

### 4. 她为什么一定要领证

冬生以为女方要领证是因为不信任他。

路飞："她要领证的本质，不是不信任你。是不信任她自己。"

两个维度：
- 她觉得自己越来越老，你会越来越不喜欢她。她不懂你需要的是什么。
- 她知道自己对你感情不深厚，她怕自己坚持不了跟你在一起。

所以她要求结婚——有了结婚证，双方都会考虑代价。她用法律约束自己。

**她的不信任，不是针对你，是针对她自己。**

### 5. 你有四次机会可以挽回

路飞分析了冬生错过的四个关键时刻：

**第一次**：她说一万块有侮辱感时。你应该给两万，然后说："如果你觉得一万不能证明我们的关系，那我以后不给了。"背对背睡一觉。第二天转两万，她就会起来给你做饭。

**第二次**：她提结婚时。你给她两万，她还要结，你给三万，然后说："我们做财产证明，我拿40%作为夫妻共同财产，剩下的是我的私人财产。我也不能保证一辈子对你一个人好。"——这样她安心，你们还能在一起。

**第三次**：她离家出走时。你不应该断她的粮。你继续给钱，她反而会回来。因为你把钱当武器控制人的时候，就是你的问题。

**第四次**：她回来后。你可以说："我给你儿子300万，你就跟着我过。如果还实在过不下去，那我们就不过了。"

**核心：你一开始就没想好，要为这段关系花多少钱、买什么单。**

### 6. 爱是放手，不是抓取

"你怕给她加满油她就跑了。如果你真的爱一个人，你就不怕给她加满油。爱一个人是希望她好，不是希望我好。"

"爱是抓不住的。如果你在他离开你的时候，你依然选择给他钱，他有富足的生活可以过——这个时候他反而会回来跟你再谈。"

"你要的是一种安全感。这种安全感是你把心交给别人赋予你的。但你又不敢把所有的心交给别人，你只敢交一部分。那么他也只敢交一部分给你。"

### 7. 冬生其力

路飞最后送了四个字：**冬生其力**。

冬天还有力量。在最冷的时候，力量反而在积蓄。

不是在春天播种才叫努力。在冬天忍耐、等待、积蓄——这本身就是力量。

## 四、提炼：人性的本质

路飞在对话中提炼出的人性核心：

**1. 人心只有两个东西：贪婪和恐惧。**
应对的就是"需求"——我们对别人的要求。

**2. 钱的本质是信任问题。**
钱的发明来源于人与人之间的不信任。当你用钱来建立关系时，你传递的是不信任。

**3. 为福选货。**
你明知道这是不好的事情，但你依然会去做——因为你想要得到反馈。你每一次借钱给别人，为的不是钱本身，而是反馈。但反馈往往是你接受不了的。

**4. 因果之间有随机性。**
种下因，不等于直接得到果。中间有"天选"——别人的选择、环境的变化、时间的推移。试图强行加速因果，代价巨大。

**5. 爱是反亏式的。**
你要的爱不是无条件的，是有来有回的。如果一个人对你无条件的爱，你是不满足的。爱必须产生巨大的矛盾，矛盾本身让爱有价值。越被试探，爱就越显得努力。

**6. 习惯不能捆绑产生。**
你一样这个人天天跟着你，不能靠绑。绑出来的不是习惯，是怨恨。习惯是在自由选择中自然形成的。

## 五、与交易框架的映射

| 人性本质 | 交易对应 |
|---------|---------|
| 强行嫁接因果 | 讼卦：证明自己对，交出定价权 |
| 用钱代替价值 | 用杠杆代替判断 |
| 怕给她加油她就跑了 | 止损恐惧：怕止损后反弹 |
| 她的不信任是针对自己 | 市场的波动是随机的，不是针对你 |
| 冬生其力 | 需卦：冬天蓄势，等待 |
| 爱是放手 | 等 = 不抓取 = 定价权在手 |
| 因果之间有随机性 | ΔQ积分看惯性，不追瞬时 |
| 为福选货 | 明知不该追，还是追了 |

**一句话：冬生在股市里懂"等"，在感情里不懂。他在K线图上能坐得住，在人心面前坐不住。**

技术透彻了，自然重视人性——但技术是在市场里练的，人性是在关系里练的。两个战场，同一套兵法。
