## 一、核心理念

### "人选祸天选福"（原句）
- **人**: 自己
- **天**: 除了自己以外的一切变量
- **选**: 当下的选择
- **祸**: 不是灾难，是代价
- **福**: 不是运气，是稳定的结构

**原话**: "人生无法决定结果，但可以选择承担什么代价。我们无法避免痛苦，也无法保证得到自己想要的。"

## 二、世界观：硬币的两面

### 正面：唯物世界（决定论）
- 世界运行在规律中，宇宙是决定论的
- 电子跃迁、晶体构造、星球运动、人类进化 = 规律的必然结果
- 时间只是打开书本翻页阅读的顺序
- 严格来说：人没有自由意志

### 反面：唯心世界
- 任何物体不真实存在，只是事物在心中的投影
- "你心中的我" ≠ 真实的我
- 唯物与唯心是同一硬币的两面，无法同时观察

### 破画原理
- 画中人无论力气多大，无法撕毁画卷
- 科学解释：观测本身就是变量的一部分
- 你无法跳出"命运"，但仍须为这一生负责

---

## 三、贪禁与恐惧（两块石头）

### 河底的石头比喻
- **贪禁（白石）**: 想拿美丽的石头 → 伸手
- **恐惧（黑石）**: 感受到寒冷退缩 → 缩手
- **借口与自洽**: 被石头搞乱的表面水流

### 核心洞察
- 人一生的决定，都在这两块石头之间
- 问题不在贪禁，也不在恐惧
- 问题在于：两者之间，还有没有可以选择的**结构**
- 这个结构 = **天权**

---

## 四、天权与五种境界

### 天权
- 贪禁与恐惧互相牵制时，人还能稳住
- 一边彻底压倒另一边 → 走向极端（病态抑郁/焦虑）

### 五种境界（由低到高）

| 境界 | 天权状态 | 特征 |
|------|---------|------|
| **迷失** | 崩塌 | 一路加码一路退缩 |
| **求福** | 摇摆 | 趋利避害，哪里有好处往哪走 |
| **见机** | 可视 | 清晰看到：选择、代价、结果 |
| **持衡** | 可撑 | 明白人生无法避免代价，主动承担选祸 |
| **两忘** | 消散 | 没有贪禁，没有恐惧，两相忘，不需要天权 |

**注意**: 境界是即时状态，人心似水随境而动。今天清醒明天可能糊涂。

## 五、人选 → 祸 → 天选

### 三个词的逻辑
1. **人选**: 你当下的选择（人 + 天 = 自己 + 其他变量）
2. **祸**: 代价（不是灾难）
3. **天选**: 结果（你只决定选择，结果不属于你一人）

### 关于"运气"
- "运气"让人停止思考、自洽
- 很多所谓运气 = 你看不见的选择叠加
- **"这个世界没有运气，只有选择"**

## 六、书的结构（据话外篇）

### 三部分
1. **人选天选人**: 通过学习人们选择中贪禁和代价的符号，建立天机，概率性判断和预测一个人/事物的内心世界。但代价巨大。
2. **人选天选天**: 探索因果账单、观察者欺骗的思考，探寻走向两忘的设想。
3. **人选天选论**: 宇宙观、世界观、价值观的完整阐述。

### 关键警告
- 反馈效率的提高和机会本身，**只能用于建立更稳定的结构、调节天权**
- 否则会在结果中付出巨大代价

## 七、与交易的关联（推导）

虽然此书不直接讲交易，但核心思想可映射：

| 书中概念 | 交易映射 |
|---------|---------|

| 恐惧 | 恐慌、杀跌、犹豫 |

| 五境界 | 交易者心理成长阶段 |
| 运气=选择叠加 | 盈亏是多重选择的结果 |
