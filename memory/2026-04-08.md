## 股票交易心得（待整理入手册）

### 1. 均线的本质 — 散户成本线而非主力成本线
- 5日线/20日线反映的是散户和中线波段资金的平均持仓成本
- 游资主力持仓周期1-3天，成本在**当日分时均价**和**封板价**，不在均线上
- 均线位置的支撑/压力 = 散户共识 → 55开，随机性大
- 游资把均线当"散户心理支撑位"，制造假突破或假跌破
- **真正跟踪游资**：分时均价线、龙虎榜席位、封单量变化、小单占比
- 均线适合判断中期趋势方向，不适合找买卖点

### 2. 趋势的本质 — 出货失败继续拉升出货（卡文斯顿）
- 趋势 = 反复出货失败
- 主力想卖但买盘更强 → 被迫在更高价位再试出货 → 又失败 → 继续涨
- 每次出货失败留下的低点 = 趋势中的回调低点
- 直到某一次出货成功 → 趋势终结
- **实战推论**：
  - 回调不破前低 = 出货失败，还得涨
  - 放量滞涨 = 出货终于成功了 = 顶部信号
  - 缩量上涨 = 根本没人想卖 = 趋势健康
- 趋势的本质：供需失衡的持续失败修正
- 与缠论"中枢"相通：中枢上移=出货失败继续拉，中枢下移=出货成功

### 3. 股东人数 — 炒作活跃度指标
- **< 2万**：太集中，庄股，流动性差，游资不敢进
- **> 20万**：太分散，散户浮筹重，拉升抛压大
- **2万-20万**：甜区 — 流动性够 + 筹码不过分散 + 游资敢进敢出
- **进阶信号**：
  - 股东人数持续减少 → 筹码集中，有人吸筹
  - 股东人数突然暴增 → 筹码分散，主力派发
  - 股东人数减少 + 股价横盘 → 最佳建仓信号
- 数据来源：东方财富F10，每季度更新
- 组合信号：股东人数减少 + 卖盘小单>80% = 庄家散户手法吸筹
