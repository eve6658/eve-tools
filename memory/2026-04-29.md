### 核心产出

**1. 语言净化原则**
- 交易术语（支撑位、压力位、价格墙）自带方向预设，污染认知
- 只用名词描述事实（"挂了3万手"），不用动词暗示因果（"强支撑"）
- 语言塑造认知 → 认知决定操作

**2. 资金流动的数学描述（牛顿式）**
- 净流量 ΔQ = 主动买 - 主动卖（唯一不可伪造的信号）
- ΔQ 积分 I(T) 看惯性（不看瞬时）
- 挂单分布熵 H 判断资金有序/无序状态
- 小单占比识别主力吸筹/出货（实盘验证：卖盘小单>80%+买盘<30%=吸筹）

**3. 交易的期权本质**
- 带止损入场 = 自制期权
- 止损金额 = 权利金，方向对的涨幅 = 行权收益
- 止损是选行权价：太窄被洗，太宽太贵
- 整个系统归结为不等式：止损×假信号频率 < 盈利×真信号频率

**4. 定价权理论**
- 定价权 ∝ 资本量（散户没有价格定价权）
- 散户唯一定价权 = 定价自己的损失（止损）
- 止损不是"割肉"，是行使自己仅有的定价权
- 放弃止损 = 放弃唯一权力 = 将命运交给市场

**5. 易经序卦与交易周期（附录）**
- 需卦（水天需）：云在天上，等待落下 → 顺应惯性，等，定价权在手
- 讼卦（天水讼）：天往上水往下，必然争 → 主观预判，证明自己对 → 交出定价权 → 亏损
- 师卦（地水师）：水藏地下，军队潜伏 → 讼卦亏损后，用纪律重建仓位管理
- 核心定理：永远不要证明自己是对的 → 不证明 → 定价权在手
