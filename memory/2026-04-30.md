## 洗盘/出货数学结构（10:22-10:38，与Adam对话总结）

### 核心纠正：大买单不是吸筹，是价格墙
- 之前错误理解：大单在低位吸筹（主力买）
- 正确理解：大单是**价格墙**，阻止散户在低位买入
- 散户想买 → 被大单墙堵在外面 → 只能追高接盘

### 洗盘结构（建仓期）
- **碎卖单（1-10手）**：主力分拆卖单，控速压价，伪装散户卖出
- **大买单墙**：不是吸筹，是**托价墙**（挡住散户的买路）
- 本质：卖的是主力（碎单控速），买盘大单是**价格墙**，散户被挤出去
- 数学结构：小卖单维持下降惯性（微分方向向下但加速度可控），大买单封锁散户操作空间

### 出货结构（派发期）— 与洗盘完全对称
- **碎买单**：主力小单拉价，制造上涨假象（控速上涨）
- **大卖单墙**：封住价格天花板，散户想卖但被堵
- 散户看到"涨了"追进来接盘 → 主力出货完成

### 洗盘 vs 出货对称表

| | 洗盘（建仓） | 出货（派发） |
|---|---|---|
| 碎单方向 | 小卖压价 | 小买拉价 |
| 墙的方向 | 大买墙封路（不让散户买） | 大卖墙封顶（阻止上涨） |
| 散户行为 | 恐惧割肉 | 贪婪追高 |
| 主力目的 | 低位拿筹码 | 高位出货 |

### 核心原则：碎单定方向，大墙封空间
- **资金是水，遇山而停**（山 = 价格墙）
- **价格墙 = 势能**（大单集中在一个价位区间）
- 墙厚 → 价格稳（势能强）
- 墙打薄 → 价格缓跌（势能减弱）→ "慢慢抽水"
- 墙撤光 → 价格突变（动能释放）
