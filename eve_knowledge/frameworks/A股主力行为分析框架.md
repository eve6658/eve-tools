# A股主力行为分析框架

> **框架来源**：2026-04-28 实盘研讨（Adam口述 + Eve整理）
> **核心思想**：不同类型的主力有不同的行为模式，必须用不同的分析方法应对
> **适用范围**：A股中小盘股（主板+科创板+创业板）

---

## 前言：一个颠覆性的认知

> **"股价经历一波拉升后，主力只有一个目的——出货。没有洗筹，没有吸筹，所有动作都是为了卖出。"**
>
> **"价格下跌到支撑位 = 出货成功的证据，不是买点。"**
>
> **"基金申购赎回窗口是油门，企业财报是涡轮增压。财报好涨得猛，财报普通也不影响涨。"**
>
> —— Adam

这两个认知颠覆了传统技术分析中"跌到支撑=买点"的逻辑。本框架以此为出发点，构建了一套完整的主力行为判别体系。

## 第一章：主力分类（四种类型）

### 1.1 游资出货型（鹭燕模式）

**代表票**：鹭燕医药（002788）

**特征**：
- 经历一波拉升后开始出货
- 盘口用"人数骗局"伪装——大量小单（每笔8-15手）制造"市场活跃"假象
- 笔数多、每笔极小，挂单集中在卖1-卖5
- 买盘存在但笔数为零（挂着看的，不成交）
- 价格从高位持续下行

**盘口识别方法**：
```
1. 卖1每笔平均手数 < 10手 → 小单密集出货
2. 买盘1-10档笔数为零 → 假单
3. 买盘突然暴增但价格不涨反跌 → 诱多陷阱
4. 卖盘/买盘档数比 > 2:1 → 供过于求
```

**分析方法**：L2盘口分析 ⭐⭐⭐⭐⭐

**操作策略**：不参与。主力在卖，你不要买。

---

### 1.2 庄家对倒型（奥瑞德模式）

**代表票**：奥瑞德（600666）

**特征**：
- 经历一波拉升后出货
- 盘口用"金额骗局"伪装——巨量托单（25,037手/1,487笔）制造"有人护盘"假象
- 每笔平均16-18手，比游资略大但仍是中小单
- 整数关口挂巨量"铁底"（5.60、5.50），但这些铁底会被击穿
- 6.39挂69,867手远端压单，封死上涨空间

**盘口识别方法**：
```
1. 整数关口出现巨量托单（>10,000手）→ 诱饵
2. 托单每笔15-20手 → 对倒特征
3. 价格往托单方向移动 → 出货成功（不是支撑）
4. 托单最终被击穿 → 陷阱确认
```

**分析方法**：L2盘口分析 ⭐⭐⭐⭐

**操作策略**：不参与。整数关口的大单是陷阱，不是支撑。

---

### 1.3 大基金锁仓型（华虹公司模式）

**代表票**：华虹公司（688347）

**特征**：
- 大基金长期持有，锁仓不动
- 盘口极度稀疏：每档恰好1-2笔，委托量个位数
- 没有对倒，没有假单，全是真实挂单
- 被动交易：挂好限价单等着成交，不主动打价格
- 价格涨跌由外部资金流（基金申购/赎回）驱动，不由盘口操控

**盘口识别方法**：
```
1. 每档笔数恰好1-2笔 → 被动交易特征
2. 委托量个位数 → 真实散户+基金拆单
3. 没有巨量托单 → 没有对倒
4. 没有笔数为零的价位 → 没有假单
```

**分析方法**：L2盘口 ⭐⭐（仅参考）+ K线/基本面 ⭐⭐⭐⭐

**操作策略**：看资金流和行业趋势。盘口没有额外信息量。

---

### 1.4 资金流驱动型（中船特气模式）

**代表票**：中船特气（688146）

**特征**：
- 同样是大基金控盘，但处于被动吸筹/增持阶段
- 盘口结构和大基金锁仓型一样（稀疏、真实）
- 价格走势由基金资金流决定：有钱进来就涨，没钱就横盘
- 企业基本面（财报）是辅助因素：财报好涨得猛，财报普通也不影响涨

**核心认知**：
```
基金申购赎回窗口 = 油门（有没有钱进来）
企业财报 = 涡轮增压（涨多快多猛）

有钱进来 + 财报好 → 涨得猛（如中船特气41→74，80%/月）
有钱进来 + 财报普通 → 也涨，慢一些
没钱进来 + 财报再好 → 涨不动
没钱进来 + 财报差 → 跌
```

**分析方法**：L2盘口 ⭐⭐ + K线 ⭐⭐⭐ + 资金流 ⭐⭐⭐⭐⭐

**操作策略**：
```
买入条件：基金申购窗口打开 / 政策利好
卖出条件：基金申购降温 / 出现赎回潮
财报作用：安全垫（不会暴雷），不决定方向
```

## 第二章：出货判别（Adam框架）

### 2.1 出货三板斧

| 手法 | 特征 | 盘口表现 |
|------|------|----------|
| **放量出货** | 高位放量，明牌砸盘 | 日成交50万手+，价格下跌 |
| **缩量出货** | 大单拆小单，隐藏卖盘 | 成交量萎缩，但笔数不减 |
| **拉升再出货** | 出货失败→拉一波→重新卖 | 先缩量止跌→突然拉高→高位震荡出货 |

**三板斧循环**：
```
放量出货 → 跌多了卖不动 → 缩量
→ 缩量出货 → 卖不动 → 拉升再出货
→ 制造反弹假象 → 在反弹区间继续出货
→ 卖得差不多了 → 不再护盘 → 再次破位下跌
```

### 2.2 出货成功 vs 出货失败

**❌ 出货成功（危险信号）**

| 信号 | 说明 |
|------|------|
| 价格从高点持续下行 | 下行本身就是出货成功的证据 |
| "铁底"被蚕食 | 买盘大单持续减少，直至消失 |
| 整数关口出现大买单 | 不是支撑，是诱散户抄底的陷阱 |
| 卖盘每个价位都有人接 | 主力想卖就能卖掉 |
| 上方压单越挂越多 | 封死上涨空间 |

**✅ 出货失败（买入机会）**

| 信号 | 说明 |
|------|------|
| 卖盘压单被主动吃掉 | 连续大单直接吃，不等回调 |
| 买盘主动追价 | 连续B大单，价格逆向上移 |
| 卖盘大面积撤单 | 压单主动消失 |
| 供需比逆转 | 买盘档数 > 卖盘档数 |

### 2.3 核心逻辑（必须记住）

```
价格往托单方向走 = 出货（主力在卖）
价格远离托单方向走 = 吸筹（主力在买）

"跌到了支撑" ≠ 买点
"跌到了支撑" = 出货成功的证据
```

## 第三章：盘口识别技术

### 3.1 两种出货伪装手法

**人数骗局（游资型）**：
- 笔数多 + 每笔极小（<10手）
- 给你看到"好多人在交易！"
- 散户看到"市场活跃"就安心

**金额骗局（庄家型）**：
- 手数大 + 托单厚（>10,000手）
- 给你看到"好大的资金在托！"
- 散户看到"有人护盘"就安心

**共同的识别方法**：
无论哪种伪装，**买盘笔数为零 = 假单**。
挂在那里的单子一笔都不成交 = 装饰品，不是支撑。

### 3.2 笔数-手数分析公式

```
每笔平均手数 = 总手数 ÷ 总笔数

< 10手/笔 → 极度碎片化，大概率伪装出货
10-30手/笔 → 中小单，可能是散户或拆单主力
> 50手/笔 → 有一定大单成分
```

### 3.3 供需比分析

```
卖盘档数 : 买盘档数
> 2:1 → 严重供过于求（鹭燕10:26: 77:35）
> 1:1.5 → 卖压沉重
≈ 1:1 → 供需平衡
< 1:1 → 需大于供（可能有资金进场）

⚠️ 但注意：买盘暴增但价格不涨反跌 = 诱多陷阱
鹭燕10:28: 买盘从35→128，但价格从14.08→14.06
```

### 3.4 笔数变化分析

```
笔数减少 → 参与者减少 → 股东人数减少
笔数归零 → 该价位所有挂单者已离场或撤单
笔数从零变为活跃 → 卖盘开始加速出逃（恐慌开始）
```

## 第四章：止损方法（20日振幅法）

### 4.1 计算方法

```
每日振幅 = (当日最高 - 当日最低) / 前日收盘 × 100%
20日平均振幅 = Σ(每日振幅) / 20
止损价 = 买入价 × (1 - 平均振幅% × 倍数)
```

### 4.2 资金量与止损策略

| 资金量 | 止损策略 | 逻辑 |
|--------|---------|------|
| 小资金（<10万） | 全20日振幅（宽止损） | 绝对金额小，亏得起 |
| 中资金（10-50万） | 近5-10日振幅 | 平衡容错空间和风险 |
| 大资金（>50万） | 近3-5日振幅（窄止损） | 绝对金额大，必须控制单笔亏损 |

### 4.3 仓位计算公式

```
最大买入股数 = (总资产 × 单次风险比例) / (买入价 - 止损价)
推荐单次风险比例：2%

实例（50万资金，买入14.15，近5日振幅4.4%止损13.53）：
  可买 = 500,000 × 2% / (14.15 - 13.53) = 10,000 / 0.62 = 16,129股
  所需资金 = 16,129 × 14.15 = 228,225元（占45.6%）
  建议1/3仓位试仓：5,300股，75,000元
```

## 第五章：资金流分析（新范式）

### 5.1 什么是资金流驱动

部分股票（尤其是大基金控盘的科创板/半导体股）的涨跌不由盘口操控决定，也不由企业财报决定，而是由**基金申购赎回带来的被动资金流**决定。

### 5.2 机制

```
政策利好 → 散户/机构申购半导体主题基金
→ 基金募到钱，必须按比例买入持仓股
→ 成分股被被动买入 → 价格推高

反之：
市场恐慌 → 基金遭遇赎回
→ 被动卖出持仓股
→ 价格下跌
```

### 5.3 如何利用资金流

```
1. 找到即将开放申购的基金（天天基金/东方财富）
2. 查看基金持仓股（前十大重仓）
3. 在基金申购日前1-2天建仓
4. 基金集中买入期（1-3天）后卖出

脚本工具：scripts/fund_flow_screen.py
```

### 5.4 财报与资金流的关系

```
基金申购赎回 = 油门（决定方向）
企业财报 = 涡轮增压（决定速度）

有钱进来 + 财报好 → 涨得猛（中船特气41→74，+80%/月）
有钱进来 + 财报普通 → 也涨，慢一些
没钱进来 + 财报再好 → 涨不动
没钱进来 + 财报差 → 跌
```

### 5.5 验证方法（内部信息优势）

如果你认识企业的员工（如财务部门），可以验证：
- 企业经营状况是否和股价涨幅匹配
- 如果经营没大变化但股价大涨 → 纯资金驱动
- 资金驱动的票 → 一旦资金退潮，回调会很快

## 第六章：日内时间节奏

### 6.1 主力的日内节奏

| 时段 | 特征 | 主力行为 |
|------|------|---------|
| 上午（9:30-11:30） | 散户最活跃、最冲动 | 主力在**出货**（利用散户热情） |
| 中午（11:30-13:00） | 休市讨论 | 主力评估出货效果 |
| 下午（13:00-14:30） | 散户松懈 | 主力可能开始**试探性买入** |
| 尾盘（14:30-15:00） | 犹豫过夜 | 制造"弱势"或"抢筹"假象 |

### 6.2 操作节奏

```
观察出货：上午看就够了（就是出货时间）
观察吸筹：下午13:00以后再看
建仓时机：下午14:00-14:30（主力试探性买入时段）
```

## 第八章：核心原则（Adam总结）

1. **主力只有一个目的：出货。** 不要用"洗筹""吸筹"来美化下跌。

2. **价格下跌到支撑位 = 出货成功的证据。** 不要以为"跌到了支撑"就是买点。

3. **整数关口的"铁底"大概率是陷阱。** 主力利用散户的整数关口心理挂假单。

4. **缩量不等于止跌。** 大单拆小单可以制造缩量假象。

5. **出货失败才是买点。** 主力想卖但卖不掉，价格自然反弹，才是底部信号。

6. **止损与资金量正相关。** 小资金可以宽止损搏反弹，大资金必须窄止损控风险。

7. **基金申购赎回是油门，财报是涡轮增压。** 财报好涨得猛，财报普通也不影响涨。

8. **主力上午卖，不吸筹。** 上午是出货时间，观察吸筹要等下午。

9. **真正活跃的盘口不会每档恰好1笔。** 太均匀 = 程序化拆单或被动交易，不是自然散户。

10. **笔数为零 = 假单。** 无论多大的单子，一笔都不成交就是装饰品。
