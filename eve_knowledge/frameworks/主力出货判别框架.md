# 主力出货判别框架（L2盘口分析）

> **核心前提**：股价经历一波拉升后，主力只有一个目的——出货。
> 没有洗筹，没有吸筹，所有动作都是为了卖出。
> 这是分析一切盘口行为的出发点。
>
> 贡献者：Adam（核心洞察）+ Eve（数据验证）

---

## 一、出货的两种伪装手法（Adam框架）

| 手法 | 鹭燕型（散户式） | 奥瑞德型（机构式） |
|------|----------------|------------------|
| **伪装维度** | 用**笔数**制造繁荣 | 用**手数**制造繁荣 |
| **表现** | 333手/**40笔**，每笔8.3手 | **24,372手**/1,327笔，每笔16.8手 |
| **给散户看到的** | "好多人在交易！" | "好大的资金在托底！" |
| **心理效果** | 市场活跃，有参与感 | 有人护盘，跌不动 |
| **适用票型** | 中小盘、散户参与度高 | 波动大、有题材 |
| **识别方法** | 笔数多但每笔极小 | 手数大但价格持续下行 |

### 核心识别公式
```
每笔平均手数 = 总手数 ÷ 总笔数
< 10手/笔 → 极度碎片化，大概率是伪装
10-30手/笔 → 中小单，可能是散户或拆单主力
> 50手/笔 → 有一定大单成分
```

### 终极判断
无论用笔数还是手数伪装，**买盘笔数为零 = 都是假的**
挂在那里的单子一笔都不成交 = 装饰品，不是支撑

## 二、出货三板斧

| 手法 | 特征 | 盘口表现 |
|------|------|----------|
| **放量出货** | 高位放量，明牌砸盘 | 日成交50万手+，价格下跌，不怕你知道 |
| **缩量出货** | 大单拆小单，隐藏卖盘 | 成交量萎缩，但笔数不减，价格横盘阴跌 |
| **拉升再出货** | 出货失败→拉一波→重新卖 | 先缩量止跌→突然放量拉高→高位震荡出货 |

### 三板斧循环模型
```
放量出货 → 跌多了卖不动 → 缩量
→ 缩量出货 → 还是卖不动 → 价格到支撑位
→ 拉升再出货 → 制造反弹假象 → 吸引抄底资金
→ 在反弹区间继续出货 → 卖得差不多了 → 不再护盘
→ 再次破位下跌 → 下一个台阶
```

## 三、出货成功 vs 出货失败（Adam框架）

### ❌ 出货成功（危险，不要买）

| 信号 | 说明 |
|------|------|
| 价格从高点持续下行 | 14.30 → 14.12 → 14.00 本身就是出货成功的证据 |
| "铁底"被蚕食 | 买盘大单持续减少（1,534手 → 962手 → 消失） |
| 整数关口出现大买单 | 14.00的1,177手不是支撑，是诱散户抄底的陷阱 |
| 卖盘每个价位都有人接 | 主力想卖就能卖掉，价格一路下滑 |
| 上方压单越挂越多 | 15.24从3,274 → 6,590 → 11,284手，封死上涨空间 |

**判断逻辑：价格跌到某个位置 ≠ 到达支撑。价格跌本身就是出货成功的证据。**

### ✅ 出货失败（机会，可以观察）

| 信号 | 说明 |
|------|------|
| 卖盘挂单被主动吃掉 | 14.30的333手被连续大单直接吃，不等回调 |
| 买盘主动追价 | 连续红色B大单，14.12 → 14.13 → 14.14 连续上移 |
| 卖盘大面积撤单 | 压单主动消失，不是被吃掉而是撤走了 |
| 价格逆向上涨 | 在没有利好消息的情况下，价格自然上移 |
| 卖盘出现断层 | 某个价位突然没有卖单，买盘真空突破 |

**判断逻辑：主力想卖但卖不掉 = 底部信号。想卖但没人接 = 真正的出货失败。**

## 四、交叉验证：小单占比框架（Eve研究）

> 参考：002788 鹭燕 4/19 实盘，卖1平均39手，买1平均152手

### 经验阈值

| 指标 | 吸筹特征 | 出货特征 | 当前鹭燕 |
|------|---------|---------|---------|
| 卖盘小单占比 | >80% | >80% | ⚠️ 卖1仅8.3手/笔，极高 |
| 买盘小单占比 | <30% | 也可能高 | ⚠️ 14.10买盘15.7手/笔，同样高 |
| 价格稳定性 | 价格稳定在窄幅 | 价格持续下行 | 🔴 14.30→14.12，下行 |
| 铁底/托单 | 托单不减反增 | 托单被蚕食消失 | 🔴 1,534→962→减少 |

### 关键区分：同样是小单，吸筹和出货的区别

```
小单占比高 + 价格稳定 + 托单增加 = 小单吸筹（主力在买）
小单占比高 + 价格下行 + 托单减少 = 小单出货（主力在卖）
小单占比高 + 价格上行 + 压单减少 = 可能是真突破
```

**结论**：小单占比只是一个维度，必须结合价格方向和托单变化综合判断。

## 五、交叉验证：价量关系框架

### 缩量出货的识别（陷阱识别）

| 正常认知 | 主力的伪装 | 如何识破 |
|---------|-----------|---------|
| 缩量 = 抛压衰竭 | 大单拆小单，总量下降但出货量不变 | 看笔数：量缩但笔数不减 = 缩量出货 |
| 地量见地价 | 地量只是因为卖单被拆散了 | 看价格能否突破：缩量但价格横盘 = 不是止跌 |
| 放量 = 有资金进场 | 放量可能是主力对倒 | 看后续：放量后价格下跌 = 放量出货 |

### 量价配合判断表

| 量 | 价 | 判断 | 操作 |
|----|----|------|------|
| 放量 | 上涨 | 可能是真突破或拉升出货 | 观察压单是否被吃掉 |
| 放量 | 下跌 | 出货无疑 | 不买 |
| 缩量 | 横盘 | 缩量出货或真正缩量止跌 | 看笔数和托单变化 |
| 缩量 | 上涨 | 量价背离，买盘虚弱 | 不追 |
| 放量 | 横盘 | 高换手，多空激战 | 等方向出来 |

## 六、止损框架：20日振幅法（Adam方法）

### 计算方法
```
每日振幅 = (当日最高价 - 当日最低价) / 前日收盘价 × 100%
20日平均振幅 = Σ(每日振幅) / 20
止损价 = 买入价 × (1 - 平均振幅%)
```

### 资金量与止损策略

| 资金量 | 止损策略 | 逻辑 |
|--------|---------|------|
| 小资金（<10万） | 可用全20日振幅（宽止损） | 绝对金额小，亏得起 |
| 中资金（10-50万） | 用近5-10日振幅（中等止损） | 平衡容错空间和风险 |
| 大资金（>50万） | 用近3-5日振幅（窄止损） | 绝对金额大，必须控制单笔亏损 |

### 仓位计算公式
```
最大买入股数 = (总资产 × 单次风险比例) / (买入价 - 止损价)
推荐单次风险比例：2%
```

### 鹭燕医药实例（2026-04-28）
- 20日平均振幅：8.17%
- 近5日平均振幅：4.4%
- 买入价：14.15
- 1x振幅止损(8.17%)：12.99（太宽，接近20日最低13.21）
- 近5日振幅止损(4.4%)：13.53（更合理）

## 七、笔数-人数关联（Adam洞察）

```
笔数减少 → 参与者减少 → 股东人数减少
```
- 笔数大幅萎缩 = 市场参与度下降 = 筹码在重新分配
- 结合价格方向判断：价格下行 + 笔数萎缩 = 散户在跑，筹码集中到少数人手里
- 笔数归零 = 该价位的所有挂单者已离场或撤单

## 八、买盘/卖盘比例（供需结构）

```
买盘档数 : 卖盘档数
> 1:1.5 → 供过于求，卖压沉重（鹭燕10:26: 35:77）
≈ 1:1 → 供需平衡
< 1:1 → 需大于供，可能有资金进场
```

**但注意**：买盘暴增但价格不涨反跌 = 新增买盘是诱多陷阱（鹭燕10:28: 买盘从35→128，但价格从14.08→14.06）

## 七、核心原则（Adam总结）

> 1. **主力只有一个目的：出货。** 不要用"洗筹""吸筹"来美化下跌。
> 2. **价格下跌本身就是出货成功的证据。** 不要以为"跌到了支撑"就是买点。
> 3. **整数关口的"铁底"大概率是陷阱。** 主力利用散户的整数关口心理挂假单。
> 4. **缩量不等于止跌。** 大单拆小单可以制造缩量假象。
> 5. **出货失败才是买点。** 主力想卖但卖不掉，价格自然反弹，才是底部信号。
> 6. **止损与资金量正相关。** 小资金可以宽止损搏反弹，大资金必须窄止损控风险。

## 十、大股东出货完整链条（2026-05-01 新增）

> **核心认知**：自始至终只有出货，没有建仓。筹码永远从大户流向散户，单向不可逆。

### 三步递进模型

| 阶段 | 动作 | 本质 |
|------|------|------|
| **1. 控制权** | 筹码最多 = 话语权 = 股价方向由我说了算 | 权力阶段 |
| **2. 推价出货** | 找有钱大哥签协议，大哥推股价，大股东趁高位出货，大哥拿利润走人 | 变现阶段 |
| **3. 跌停出货** | 维护股价需要资金成本，跌停直接砸给韭菜，成本更低 | 清仓阶段 |

### 关键认知纠偏

- ❌ 错误：跌停是因为推不动
- ✅ 正确：跌停是成本更低的出货方式，推完高位后的主动选择
- ❌ 错误：有新主力建仓
- ✅ 正确：自始至终只有卖，所谓"新主力"是同一个卖方换马甲
- ❌ 错误：散户追涨是马丁法则
- ✅ 正确：散户**下跌补仓**才是马丁法则，追涨是跟风本能

### 散户行为模式

```
涨了 → 跟风追涨（本能）
跌了 → 马丁补仓（摊低成本的幻觉）
不管涨跌 → 散户都在接货 → 主力横竖都能出
```

### 出货链条的循环

```
大哥接盘 → 下一个出货周期的萌芽
→ 大哥筹码变多 → 大哥变成新大股东 → 大哥也要出货
→ 循环往复，没有终局
```

## 十一、入场时机数学模型（2026-05-01 新增）

> **核心**：用数学算哪个时期入场最优，三个维度——P(安全) × R(收益) × L(流转)

### 三时期对比

| 时期 | P(安全) | R(收益) | L(流转) | 特点 |
|------|---------|---------|---------|------|
| 下跌期 | 0.2 | 0.1 | 0.2 | ❌ 最差，送命 |
| 横盘期 | 0.7 | 0.25 | 0.6 | ⚠️ 确定但慢 |
| 推动期 | 0.3 | 0.5 | 0.7 | ⚠️ 快但险 |

### 关键：风险收益的非对称性

**止损线：每次最多亏3%**

| 策略 | 7次亏×3% | 3次赚×幅度 | 净收益 | 期望/次 |
|------|----------|-----------|--------|--------|
| 横盘期 | -21% | 3×25%=+75% | +54% | +7.7% |
| 推动期 | -21% | 3×100%=+300% | +279% | +39.9% |

### 资金效率（R÷T，T=持有时间）

| 时期 | R÷T | 说明 |
|------|-----|------|
| 下跌期 | 0.100 | ❌ 最差 |
| 横盘期 | 0.063 | 确定但周转慢 |
| 推动期 | 0.150 | 效率最高 |

### 10次交易模拟

| 策略 | 总收益 | 亏损次数 | 净赚 |
|------|--------|----------|------|
| 10次横盘 | +2.5 | 3次亏 | **+1.6** |
| 10次推动 | +1.5 | 7次亏 | **-4.0** |

**但加止损后（亏损限定3%）：**

| 策略 | 7次亏×3% | 3次大赚 | 净赚 |
|------|----------|---------|------|
| 推动期 | -21% | +300% | **+279%** |

### 最终结论

| 时期 | 适用条件 |
|------|----------|
| 下跌期 | ❌ 永远不进 |
| 横盘期 | 适合保守资金，确定性高但收益有限 |
| **推动期** | **适合有止损纪律的资金，期望收益最高** |

**核心逻辑：用3%的确定性亏损，换100%的不确定性收益。止损到位，长期必赚。**

## 十二、利夫摩尔模型（2026-05-01 新增）

> 数学推导与一百年前的经验总结殊途同归

| 利夫摩尔原则 | 数学对应 |
|--------------|----------|
| 利润是坐出来的 | 推动期持有不动，R=100% |
| 亏损是确定的 | 止损3%是成本，P=1.0 |
| 盈利是不确定的 | 100%收益是概率事件，但E(V)极大 |
| 赚钱靠等待 | 横盘期等待，推动期出手 |
| 永远不要摊平亏损 | 马丁法则=散户死法 |
| 小亏大赚 | 3%止损换100%收益，非对称性 |
