# 速查卡片：固定资产贷款全流程管控

> 一页纸，各阶段核心检查项

---

## 贷前

| 检查项 | 核心要求 |
|--------|---------|
| **资本金来源** | 非债务性资金，进入"所有者权益"科目 |
| **资本金到位** | 先于贷款到位或按比例同步 |
| **货币资本金** | 验资报告 + 银行入账凭证 + 发票 |
| **非货币资本金** | 权属清晰 + 估价合理 + 验资真实 |
| **项目合规性** | 四证齐全（用地规划、不动产权、工程规划、施工许可） |
| **项目总投** | 土建成本 + 设备成本 + 投资水平合理性 |
| **市场分析** | 产业链位置 + 上下游 + 行业龙头 |
| **建设必要性** | 新建=优势分析；扩建=产能利用+订单 |

---

## 贷中

| 检查项 | 核心要求 |
|--------|---------|
| **四证审核** | 双人核实原件，共同前往发证机关实地核查 |
| **要件一致性** | 发证日期、面积、施工单位与施工许可证一致 |
| **环保核验** | 放款当日查征信系统/企查查，截图留存 |
| **抵质押** | 土地使用权或在建工程抵押，首次不具备条件的需承诺 |
| **放款方式** | 受托支付，5个工作日内完成 |
| **支付管控** | 严禁一次性提用；交易对手在我行开户；严禁挪用 |
| **资本金台账** | 填写《项目资本金到位台账》 |

---

## 贷后

| 检查项 | 核心要求 |
|--------|---------|
| **资金流向** | 逐笔监控，3月内提供支付凭证及发票 |
| **异常监测** | 同名频繁划转、小额拆分、支付后挪用 |
| **收入归集** | 销售/租金/回款 → 归集至我行回款账户 |
| **施工进度** | 双人现场核验 + 影像留存 |
| **施工匹配性** | 原材料出入库 vs 支付合同，价格 vs 市场行情 |
| **还款设置** | 前/中/后期分期，均衡还款，引导提前还款 |
| **半年重检** | 实际运行 vs 尽调预期，偏离大则审慎评估 |
| **台账上传** | 附件形式上传至贷后定检系统 |

---

## 红线速记

```
❌ 严禁一次性提用全部贷款额度
❌ 严禁贷款资金挪用至其他项目或被集团归集
❌ 合规性要件不得后置
❌ 资本金不得为债务性资金
❌ 不得以任何理由不采用受托支付
```

---

## 数字速记

| 数字 | 含义 |
|------|------|
| **四证** | 用地规划许可、不动产权证、工程规划许可、施工许可 |
| **5工作日** | 贷款发放后完成受托支付 |
| **3个月** | 放款后提供支付凭证及发票 |
| **半年** | 项目运行重检频率 |
| **双人** | 放款审核、现场核查均需双人 |

---

## 关键文件清单

| 文件 | 阶段 | 用途 |
|------|------|------|
| 验资报告 | 贷前 | 证明资本金真实 |
| 银行入账凭证 | 贷前 | 证明货币资本金到位 |
| 四证原件 | 贷中 | 合规审核 |
| 施工许可证 | 贷中 | 核对承包人一致性 |
| 抵押登记文件 | 贷中 | 办妥担保 |
| 用款计划 | 贷中 | 明确资金用途 |
| 工程进度报告 | 贷中 | 监理单位出具 |
| 支付凭证及发票 | 贷后 | 3个月内提供 |
| 竣工验收报告 | 运营 | 一次性 |
| 项目资本金到位台账 | 全程 | 贷后定检上传 |
