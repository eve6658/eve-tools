## 核心原则（贯穿全文）

1. **资本金必须是非债务性资金**，项目主体不承担还本付息责任
2. **资本金先于贷款到位**或按投放比例同步到位
3. **受托支付**为原则，贷款发放5个工作日内完成
4. **资金封闭管理**：贷款仅用于指定项目，严禁挪用
5. **四证齐全**：用地规划许可、不动产权证、工程规划许可、施工许可
6. **双人现场核查**：放款审核、施工进度核验均需双人到场
7. **半年重检**：项目运行情况每半年重检一次
8. **台账管理**：全流程监控台账上传至贷后定检系统

## 贷款生命周期速览

```
贷前调查 → 贷中审核/放款 → 贷后监控 → 项目运营 → 还款结清
   ↓            ↓              ↓           ↓
 资本金       四证审核       资金流向     达产率
 合规性       受托支付       施工进度     收入预测
 总投合理性   抵质押办理     现场抽查     盈亏平衡
```
