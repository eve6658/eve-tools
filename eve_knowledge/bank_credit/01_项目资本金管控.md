# 01 项目资本金管控要点

## （一）项目资本金相关规定

固定资产投资项目资本金 = 项目总投资中由股东认缴的出资额 = **非债务性资金**

### 出资规则

- 股东可按出资比例依法享有所有者权益，可转让，**不得抽回**
- 可用**货币**出资，也可用**实物、产权、技术、土地使用权**作价出资

### 五条红线

| 序号 | 要求 |
|------|------|
| 01 | 资本金应进入"所有者权益"科目，体现在"实收资本"或"股本"，来源要有明确依据和证明 |
| 02 | 资本金必须是非债务性资金，项目主体**不承担任何还本付息责任** |
| 03 | 防范通过虚增利润进行股东分红、虚构债权债务关系等**抽逃资本金**行为 |
| 04 | 关注关联交易，防范股东通过非公允或虚构贸易合同**转移资本金** |
| 05 | 资本金应**先于贷款投放全部到位**或按贷款投放比例同步到位（以批复为准） |

> ⚠️ 项目资本金是固定资产贷款方案的**核心内容之一**

---

## （二）贷前阶段的尽职调查

### 货币资本金的真实性调查

调查资料：
- 验资报告
- 资金到位证明（银行入账凭证等）
- 设备及原材料购置发票、交易合同
- 银行对账单

分析重点：
- 已到位资本金的真实性
- 如借款人已有在建项目 → 分析所有者权益中扣除在建项目占用后**可用于本项目的资本金**

### 非货币资本金的真实性调查

三个调查维度：

| 维度 | 内容 |
|------|------|
| 权属清晰性 | 权属证明文件完整、真实、有效 |
| 估价合理性 | 非货币资本金的估价是否合理 |
| 验资报告真实性 | 验资报告本身是否真实 |

---

## （三）贷中阶段的审核要点

### 资本金来源及比例的合规性

分析四个来源：
1. 股东投入来源
2. 借款人积累资金的来源
3. 政府补贴来源（如有）
4. 股东借款作为来源的合规性

### 建立资本金到位及使用台账

- 收集资本金到位证明材料
- 填写《项目资本金到位台账》
- 详细记录资本金到位情况
- 收集资本金的使用证明材料

---

## （四）贷后阶段的跟踪管理

| 跟踪方式 | 具体内容 |
|---------|---------|
| **财务报表** | 了解其他应收款、长期投资、长短期借款科目中是否存在与股东或关联公司往来形成的资产 |
| **关联交易** | 关注关联交易情况，通过主营销售收入、应收账款、银行存款等明细账户确认 |
| **现场走访** | 实地走访，现场查看非货币资本金情况，与贷前调查资料比对 |
| **后续到位可靠性** | 动态评估股东资信与出资能力，结合经营与财务情况分析追加资本金的能力与意愿 |
