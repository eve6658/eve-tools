# 股东人数选股 — 记忆宫殿锚点

## 核心意象：🎪 马戏团的帐篷

想象一个马戏团帐篷（股东人数）：
- **太小**（<2万人）：观众席空荡荡，马戏团演不起来，门票卖不出（流动性差）
- **太大**（>20万人）：观众挤成一团，互相踩踏，台上的人根本没法演（抛压重）
- **刚刚好**（2-20万人）：气氛热烈，门票好卖，马戏团愿意来（游资敢进敢出）

股东人数在减少 = 观众在散场 = 马戏团（主力）在清场准备大戏。

## 空间位置：选股清单 · 股票名称旁

把这张卡放在选股清单旁边——每次F10查股东人数用帐篷比喻快速判断。

## 一级逻辑（3点）

1. 2-20万是活跃度甜区——太少没流动性，太多拉不动
2. 股东人数减少 = 筹码在集中 = 有人在准备
3. 股东人数暴增 = 筹码在分散 = 主力在派发

## 深度思考题

1. 一只股票股东人数从15万降到8万但股价没动，说明什么？
2. 股东人数从哪里查？更新频率是多久？
3. 为什么游资不敢进股东人数<2万的股票？
