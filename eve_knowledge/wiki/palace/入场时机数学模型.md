# 入场时机数学模型 — 记忆宫殿锚点

## 核心意象：🎰 赌场的期望值

想象你在赌场：
- **下注100元，输了亏3元（止损），赢了赚100元**
- 赢的概率30%，输的概率70%
- 期望值 = 0.3×100 - 0.7×3 = **+27.9元/次**
- 长期玩下去一定赚 — 这就是非对称性的数学本质

赌场不怕你赢，怕你不来。你就是散户眼中的"赌场"。

## 空间位置：计算器旁边

把这张卡放在计算器旁边——每次想交易时先算一下期望值。

## 一级逻辑（3点）

1. **P×R×L三因子** — 安全×收益×流转，综合评估入场时机
2. **非对称性** — 亏损有底（止损3%），盈利无顶（100%+）
3. **期望值为正** — 只要E(V)>0，长期玩必赚

## 三时期决策表

| 时期 | 一句话 | 操作 |
|------|--------|------|
| 下跌期 | 送命 | ❌ 永远不进 |
| 横盘期 | 确定但慢 | ⚠️ 保守资金可入 |
| 推动期 | 快但险 | ✅ 有止损纪律者首选 |

## 关键公式

```
期望收益 = P(赢)×R(赢) - P(亏)×L(亏)
推动期 = 0.3×100% - 0.7×3% = +27.9%（每次）
横盘期 = 0.7×25% - 0.3×3% = +16.6%（每次）
```

## 决策检查清单

- [ ] 止损线设好了吗？（没设 = 不要入场）
- [ ] 期望值是正的吗？（算了才知道）
- [ ] 我是在推动期还是横盘期？（决定收益上限）
- [ ] 我有止损纪律吗？（没有 = 你在用马丁法则）

## 深度思考题

2. 为什么横盘期"看着慢"但实际上是最安全的？
