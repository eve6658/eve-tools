# 大股东出货三步法 — 记忆宫殿锚点

## 核心意象：🏦 银行金库的三道门

想象一个银行金库：
- **第一道门**：金库钥匙在谁手里（控制权）
- **第二道门**：找人来搬金子（推价出货）
- **第三道门**：金子太多搬不完，打折卖给路人（跌停出货）

三道门是递进关系，不是并列。过了第一道门才能开第二道，第二道没走完才开第三道。

## 空间位置：交易桌 · 保险箱旁边

把这张卡放在交易桌上，像一个小型保险箱模型——每次做决策前摸一下，提醒自己：**筹码在谁手里？**

## 一级逻辑（3点）

1. **控制权 = 筹码最多 = 话语权** — 他说了算，你说了不算
2. **推价出货 = 找大哥合作** — 大哥推股价，大股东出货，散户接盘
3. **跌停出货 = 成本最低的收尾** — 维护股价要花钱，跌停直接砸，省成本

## 二级逻辑（5点）

1. 自始至终只有卖，没有买。所谓"新主力"是幻想
2. 散户涨了追、跌了补，横竖都在接盘
3. 大哥接盘 = 下一个出货周期的起点，循环不息
4. 散户只会跟风不会思考 — 这是整个链条成立的基础
5. PE兜底：连续跌停后PE还是10-20倍，大股东不亏

## 决策检查清单

做任何买入决策前，问自己：

- [ ] 这只票有没有经历过一波拉升？（有 = 可能在出货周期）
- [ ] 当前是三步法的哪个阶段？（控制权/推价/跌停）
- [ ] 我是不是在接盘？（散户 = 永远是出货源头）
- [ ] 有没有止损纪律？（没有 = 不要碰推动期）

## 深度思考题

1. 如果大股东有控制权，你凭什么认为自己能赢？
2. "新主力建仓"的铁底信号，有没有可能是同一个卖方换马甲？
3. 马丁补仓为什么是散户的死法？
