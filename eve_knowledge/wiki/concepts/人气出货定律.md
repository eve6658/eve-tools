# 人气出货定律

> 人气是股票出货的标志。分时盘口人气上来马上出货。

## 核心规律

人气信号与主力行为的对应关系：

| 人气信号 | 实际含义 |
|----------|---------|
| 股吧/论坛突然爆讨论 | 主力已经在出，散户刚看到 |
| 涨幅榜上榜 + 连续冲板 | 最后一波散户冲进来 → 正好出货 |
| 分时量突然放大（脉冲式） | 散户的买单 = 主力的卖单 |
| 身边朋友开始推荐 | 你已经是最后一批了 |

## 时间线

```
主力吸筹（没人关注）
→ 拉升（小部分人注意）
→ 冲板（媒体开始报道）
→ 人气爆棚（散户蜂拥）→ 主力在出货
→ 人气消退（散户套牢）→ 趋势终结
```

## 判断公式

- 你听到有人讨论这只票的时候 = 主力已经在卖了
- 你自己发现这只票的时候 = 可能还能买
- 你不得不知道这只票的时候 = 已经晚了

## 反向应用

> 一只票根本没人关注 = 主力可能正在吸筹

"股东人数减少 + 缩量横盘" = 最佳建仓信号 — 没人气 = 主力在暗处收集
