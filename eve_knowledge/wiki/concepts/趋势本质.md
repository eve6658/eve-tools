# 趋势本质

> 趋势就是出货失败继续拉升出货。K线上涨是一次一次出货失败推高的。——卡文斯顿

## 核心观点
- 趋势 = 反复出货失败
- 主力想卖 → 买盘更强 → 被迫在更高价位再试出货 → 又失败 → 继续涨
- 每次出货失败留下的低点 = 趋势中的回调低点
- 直到某一次出货成功 → 趋势终结

## 实战推论
| 信号 | 含义 |
|------|------|
| 回调不破前低 | 出货失败，还得涨 |
| 放量滞涨 | 出货终于成功了 → 顶部信号 |
| 缩量上涨 | 根本没人想卖 → 趋势健康 |

## 与缠论的关系
- 缠论"中枢" = 主力反复出货的区域
- 中枢上移 = 出货失败继续拉
- 中枢下移 = 出货成功

## 本质定义
趋势的本质：**供需失衡的持续失败修正**
