# 交易复盘方法

> 成功操作找模式，失败操作找原因，然后记录。

## 复盘评分维度（参考2026-03-30 600666复盘）

| 维度 | 问题 |
|------|------|
| 买入时机 | 为什么选这个点？是冲动还是计划？ |
| 卖出时机 | 是主动还是被动？有无提前计划？ |
| 仓位管理 | 比例是否合理？是否有止损？ |
| 情绪状态 | 操作时的心态（贪婪/恐惧/平静） |
| 信息质量 | 做决定时的信息是否准确完整？ |

## 成功操作的共同模式

1. **有明确支撑位买入**（MA5/关键价位）
2. **缩量回踩时入场**（不是追涨）
3. **大盘恐慌次日反弹时加仓**
4. **卖出有逻辑**（逐笔分析/盘口压力/分时识别主力出货）
5. **分批操作**（2/3锁利 + 1/3让利润奔跑）

## 失败操作的共同模式

1. **在"稳定"时入场** → 主力派发
2. **没有止损纪律** → 小亏变大亏
3. **单票持仓过重** → 无处腾挪
4. **追涨买入** → 进在高位
5. **盈亏比差** → 下跌空间大，上涨空间小
