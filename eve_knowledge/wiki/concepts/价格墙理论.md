# 价格墙理论

> 通过盘口委托单的堆积结构判断主力位置和趋势方向。

## 核心定义

**价格墙**：在某个价位上，委托量异常堆积形成的"墙"。可以是买单墙（支撑），也可以是卖单墙（压力）。

**关键指标**：委托量、笔数、均笔手数、价格墙的移动方向。

## 三大定律

### 一、谁卖谁定价

> 市场是由卖家决定的。主力在哪个价位挂卖单，价格就到不了那里。

不是买盘不够强，是卖盘由主力控制。主力想让价格到哪，就在哪挂卖单。价格天花板是主力画的，不是市场形成的。

**实战推论**：
- 卖单撤掉 = 主力不想压了 = 价格将由买方决定 = 行情启动
- 卖单增加 = 主力在加压 = 价格上不去
- 卖单不变 = 区间行情延续

### 二、价格墙位置 = 主力底牌

> 价格墙的位置就是主力的成本锚点，墙在哪，主力就在哪。

**买单墙（托单）**：
- 大量买单堆积在某个价位 = 主力在此建仓/护盘
- 托单规模越大，主力在此价位的决心越强
- 例：5.90处17万手买单 = 主力在5.90有约1亿元的防守意愿

**卖单墙（压单）**：
- 大量卖单堆积在某个价位 = 主力在此出货/压制
- 压单规模越大，主力在此价位的控制力越强

### 三、价格墙移动 = 趋势方向

> 不用看均线，不用看MACD。盯住价格墙的脚印，主力走到哪，趋势就到哪。

| 墙的移动 | 含义 | 判断 |
|----------|------|------|
| 墙往上移 | 主力抬高成本，看好后市 | 看涨信号 |
| 墙往下移 | 主力压低吸筹，还要更低 | 还没到底 |
| 墙不动 | 主力在区间内做T | 区间行情，低买高卖 |
| 墙突然消失 | 主力撤单，可能要变盘 | 关注突破方向 |

## 价格墙与趋势本质的关系

价格墙理论是「趋势本质」的盘口具象化：

- 趋势 = 反复出货失败推高 = 价格墙不断上移
- 趋势终结 = 出货成功 = 价格墙不再上移或开始下移
- 盘整 = 价格墙不动 = 主力在固定区间内收割

## 价格墙与主力拆单

**均笔手数不是铁律。** 主力可以拆单伪装散户。

判断主力行为需要**价格 + 委托量 + 笔数 三者联动**：

| 场景 | 均笔 | 价格 | 委托量 | 判断 |
|------|------|------|--------|------|
| 拆单吸筹 | 小（<20手） | 稳定 | 持续堆积 | 主力拆单 |
| 真散户恐慌 | 小（<20手） | 急跌 | 快速消失 | 真散户 |
| 造势挂单 | 大（>50手） | 不动 | 原地不动 | 主力造势，非真交易 |

**核心原则**：单看均笔会误判。主力最擅长把自己的单子伪装成散户。

## 逐笔成交辅助判断

**逐笔成交的笔数是个位数还是两位数，是散户和主力的分界线之一。**

- 逐笔成交每笔 **1-20手** → 散户买卖的标志（个位数到20手是散户区间）
- 逐笔成交每笔 **>20手** → 有主力/机构参与
- 逐笔大量1-20手集中出现 → 散户在集中交易，主力可能在旁边观察

配合价格墙：如果价格墙处的逐笔成交全是小单个位数 = 散户在墙两侧换手，主力挂单不动。如果逐笔出现两位数大单主动吃掉价格墙 = 主力开始行动。

## 价格墙与做T模式

主力在价格墙之间做T收割：

```
底部买单墙（5.90）→ 吸筹
        ↕ 0.1元价差
顶部卖单墙（6.00）→ 派发
```

反复循环，稳定获利。当0.1元的价差主力看不上了 = 要有更大行情。

## 实战要点

1. **盯住最大委托量的价位** — 那就是当前的价格墙
2. **观察价格墙的移动** — 上移=看涨，下移=看跌，不动=区间
3. **关注卖单墙的撤单** — 撤单=变盘信号
4. **价格墙规模变化** — 增加=主力加仓，减少=主力撤退
5. **逐笔成交笔数** — 个位数=散户，两位数以上=主力露面
