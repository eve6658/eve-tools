# 股东人数选股

> 股东数量2万到20万之间是股票可炒作活跃度指标。

## 核心区间

| 股东人数 | 状态 | 含义 |
|----------|------|------|
| < 2万 | 太集中 | 庄股，流动性差，游资不敢进 |
| **2万-20万** | **甜区** | 流动性够 + 筹码不过分散 + 游资敢进敢出 |
| > 20万 | 太分散 | 散户浮筹重，拉升抛压大，没人愿意抬轿 |

## 进阶信号

| 信号 | 含义 |
|------|------|
| 股东人数持续减少 | 筹码集中 → 有人在吸筹 |
| 股东人数突然暴增 | 筹码分散 → 主力在派发 |
| 股东人数减少 + 股价横盘 | 最佳建仓信号——吸筹但没涨 |

## 组合信号
- 股东人数减少 + 卖盘小单>80% = 庄家用散户手法偷偷吸筹
- 股东人数减少 + 缩量横盘 = 主力控盘蓄势

## 数据来源
- 东方财富F10，每季度更新（随财报披露）
