# 交易手册知识点总汇（截至 2026-04-19）

> 本文档是所有已验证知识点的完整索引，用于编写/更新正式交易手册。

## 第一部分：盘口微观分析

### 第一章：工具中性原则
- 工具（MACD/RSI/均线）不预判未来，只描述当前状态
- 用"如果...那么..."代替"所以...会..."
- 阻力位是观察区，不是卖出点
- 买卖逻辑必须对称：跟随买入，跟随卖出

### 第二章：小单占比阈值（2026-04-02 实盘量化）
- **吸筹信号**：卖盘小单 >80% + 买盘小单 <30% + 价格区间稳定 + 铁底未动
- **出货信号**：买盘小单 >80% + 卖盘大单密集 + 价格区间下移
- **散户主导**：买卖盘小单占比均衡（各50-60%），无明显大单
- **对倒信号**：买卖盘同时出现大单，金额接近，快速交替
- 数据来源：600666奥瑞德（卖盘平均39手 vs 买盘平均152手）、002788鹭燕（尾盘买盘平均165手）
- **实操四步判断法**：
  1. 看逐笔成交的买卖比
  2. 统计买/卖盘小单占比
  3. 对照价格区间是否稳定
  4. 检查铁底（万手托单）是否存在

### 第三章：价格墙理论（2026-04-13 认知升级）
- **价格墙** = 某价位堆积万手级别挂单（压单=天花板，托单=铁底）
- **核心机制**：控制者在现价上下挂大单，中间区间是散户投票区
- **反向操作**：人多→撤地板下跌洗盘 | 人少→撤天花板喷发
- **30%止损率**：马丁策略+长持者不会被洗，正好洗30%摇摆筹码
- **突破信号**：价格墙被主动吃掉 / 被主动撤掉 + 放量确认
- **价格墙厚度变化比价格本身更重要**：
  - 25万→10万→5万→消失 = 主力撤退/准备发车
  - 5万→10万→25万 = 主力加压/准备出货

### 第四章：成交量的本质（2026-04-08 认知更新）
- 成交量 = 分歧指标，不是方向信号
- 放量只能说明买卖双方想法不一样，不知道谁对
- **底部无量涨停** = 没人卖，真强
- **底部放量涨停** = 有人卖，可能对倒，观察次日
- **高位放巨量冲高回落** = 主力大规模派发，危险
- **高位缩量下跌** = 没人接盘，更危险
- T+1单边市场：5000万进去涨10%=500万浮盈，所有人都想逃，卖压必然涌现

### 第五章：千档盘口分析方法
- 必须用本地OCR（skills/local-ocr/ocr_logic.py），禁止用视觉模型
- 关注：委比变化、买/卖盘总量对比、大单分布
- 委比 = 哪边人多 → 人多的那边要被反向收割

## 第二部分：主力行为分析

### 第六章：主力操盘规则（A/B/C三种手法）
- **A手法（单边出货）**：拉高→封板→消耗封单→次日低开（鹭燕模式）
- **B手法（两头剥皮）**：吸筹→出货→回补→再拉→再出（奥瑞德模式，片皮鸭）
- **C手法（暴力拉升）**：连续涨停→高位横盘出货→阴跌（妖股模式）
- 6.40是奥瑞德的分界线：以上卖多于买（出货区），以下买多于卖（吸筹区）

### 第七章：主力拉高出货识别（2026-04-16 完整验证）
- **完整链条**：封单消耗 → 集合竞价未涨停 → 盘口委比翻负 → 超大单净流出 → 逐笔卖:买比例恶化 → 价格崩塌
- **关键信号**：
  - 集合竞价未涨停 = 出货确认第一步
  - 委比从正转负 = 诱多结束
  - 超大单4分钟净流出 >1000万 = 实锤
  - 逐笔卖:买 >3:1 = 加速出货
- **出货区间下沿 ≠ 支撑位**（Adam纠正的重要认知）

### 第八章：主力控盘度判断
- **控盘甜区**：流通盘的30-50%
- **80%控盘**：理论上利润最大，但现实中几乎不可能（举牌线、减持新规限制）
- **50%控盘**：实操中最灵活，进退有余
- **控盘度判断方法**：
  - 股东人数减少 + 人均持股量增加
  - 成交量萎缩但价格稳定
  - 极度缩量（日成交<常态的1/3）

### 第九章：主力拉升动力分析框架（2026-04-02 仁和药业案例）
- **拉升三条件**：
  1. 筹码清洗完毕（极度缩量=筹码锁定）
  2. 板块有风（板块轮动主线）
  3. 大盘配合（稳定或上涨）
- **拉升历史规律**：极度缩量→突然放量涨停→阴跌出货
- **主力成本估算**：横盘区均价 ≈ 主力成本

## 第三部分：选股与择时

### 第十章：股东人数选股（2026-04-19 更新）
- **甜区**：2万-20万人
  - 2-5万人：高度集中，主力控盘迹象明显
  - 5-10万人：中等集中，主力建仓中期
  - 10-20万人：开始分散，主力可能在派发
  - >20万人：散户化，基本没戏
  - <2万人：筹码太集中或僵尸股
- **深交所（0开头）**：每10个交易日更新一次（一个月2-3次）
- **上交所（6开头）**：只在季报/年报披露（一个季度1次）
- **最佳信号**：股东人数连续2-3期递减 + 人均持股量上升 + 股价低位横盘
- **注意**：数据有10天滞后，需配合量价关系判断

### 第十一章：价格区间选股
- **散户最佳机会窗口**：10-50元
- **5元以下**：低价股，流动性差，盈亏比低，主力不愿碰
- **100元以上**：不是散户机会，看都别看
- **主力炒作标的变迁**：以前炒2-3元→6-8元，现在炒10-20元→40-50元

### 第十二章：趋势跟踪三步法
1. **观察资金在哪**：逐笔成交买卖比、大单集中方向、小单占比
2. **等待价格墙突破**：万手压单被吃掉/撤掉 + 放量
3. **倒车离场，发车再入**：不猜顶，不赚最后一个铜板
- **三不原则**：不猜底、不猜顶、不贪心
- **铁律：涨停开盘必走**（涨停=流动性最好的出货窗口）

### 第十三章：盈亏比与仓位管理
- **最优仓位** = (胜率×赔率 - 败率) / 赔率（凯利公式）
- 凯利公式实际使用取半（保守）
- 盈亏比不达标（<2:1）不进场
- 高盈亏比只在恐惧和不耐烦中产生

## 第四部分：操盘手理论（庄家视角）

### 第十四章：控盘坐庄模型推演（2026-04-19）
- **3-15-10形态**：经典妖股套路
  - 3→5：试盘+吸筹（缓慢放量）
  - 5→10：主升浪（连续涨停，制造人气）
  - 10→15：加速赶顶（最疯狂，吸引散户接力）
  - 15→10：高位派发（横盘震荡出货）
- **50%流通控盘是实操甜区**：
  - 外面50%筹码中，真正活跃的只有15-20%
  - 实际浮动筹码只有总股本的7-10%
  - 用1-2亿就能拉动股价
- **大资金交易成本**：
  - 佣金：万1（散户万2.5）
  - 一个来回合计约千1.3（散户千3.5）
  - 大资金成本不到散户一半
- **庄家利润三来源**：筹码优势 + 成本优势 + 信息优势
- **出货阶段是最难的**：你在高位卖，必须有人在高位买
- **幸存者偏差**：看到的妖股都是走出来的，走不出来的K线上根本不会出现3-15-10

### 第十五章：妖股形态的必要条件
1. 绝对控盘（>60%，实操中30-50%更常见）
2. 没有大资金对手盘
3. 市场情绪配合（牛市或题材热点）
4. 出货阶段有足够的散户涌入
- 四条缺一条，形态就走不出来

## 第五部分：心智纪律

### 第十六章：无我交易
- **核心测试**：如果今天空仓，我会以当前价买入吗？
- 如果答案是否定的 → 不要因为持仓而持有
- 沉没成本谬误：已亏的钱不是钱，未来的盈亏才是决策依据

### 第十七章：震荡市纪律
- 追跌杀涨（不是追涨杀跌）
- 恐惧时买入，兴奋时卖出

### 第十八章：周末效应
- 周末前游资通常减仓避险
- 周五不追高，除非板块确定性强
