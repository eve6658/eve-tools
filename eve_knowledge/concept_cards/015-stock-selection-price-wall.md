# 选股因子：价格墙突破

## 核心逻辑
> **价格墙是主力的成本线和意图线。突破价格墙 = 主力发车信号。**

---

## 什么是价格墙

价格墙 = 某个价位堆积了**万手级别**的挂单（压单或托单）

- **卖盘价格墙**：万手压单在某个价位 → 主力设置的天花板
- **买盘价格墙**：万手托单在某个价位 → 主力设置的铁底

**600666 案例（2026-04-09）：**
- 7.15 → 25.4万手卖盘价格墙（天花板）
- 6.50 → 3.95万手买盘价格墙（铁底）
- 7.00 → 2.53万手卖盘价格墙

---

## 选股条件

### 发车信号（入选条件）
1. **价格墙被主动吃掉** → 万手压单被连续大单买穿
2. **价格墙被主动撤掉** → 主力自己撤单，说明不想压了
3. **突破后回踩不破** → 价格回到价格墙位置但不跌破

### 筛选标准
| 条件 | 标准 |
|------|------|
| 价格墙厚度 | ≥1万手 |
| 突破方式 | 主动买盘吃掉，不是小单磨掉 |
| 突破后量能 | 放量（比前5日均量大50%+） |
| 价格墙位置 | 距离当前价<3%（近在眼前） |

### 排除条件
- 小单磨掉的价格墙（主力没动手，散户慢慢吃的）
- 突破后立刻缩量（假突破）
- 价格墙太远（>5%距离，短期不会触及）

---

## 操作节奏

```
发现价格墙 → 加入观察列表
     ↓
价格墙开始变薄（万手→几千手）→ 准备
     ↓
价格墙被吃穿 + 放量 → 发车，买入
     ↓
价格到下一个价格墙 → 观察是否继续突破
     ↓
价格墙突破失败 + 委比恶化 → 倒车，离场
```

---

## 实战要点

**价格墙的厚度变化比价格本身更重要：**
- 25万手 → 10万手 → 5万手 → 消失 = 主力在撤退，准备发车
- 5万手 → 10万手 → 25万手 = 主力在加压，准备出货

**价格墙消失的两种情况：**
1. 被买走（主动吃掉）→ 真突破
2. 被撤掉（主力撤单）→ 可能是诱多，需要结合量能判断
