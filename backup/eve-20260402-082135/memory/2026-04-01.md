### 核心教训：买入跟随 vs 卖出预判 = 逻辑矛盾

**问题**：
- 买入时用"支撑位跟随"逻辑（到了就敢买）
- 卖出时用"阻力位预判"逻辑（分析说要出货就提前跑）
- 两个动作的底层逻辑不一致

**修正**：
- 支撑位和阻力位的用法必须对称
- **支撑位 = 观察企稳后买入**
- **阻力位 = 观察是否突破，而非预判顶部**
- 阻力区应该做的是：观察 → 突破则持有，受阻回落则走

### 阻力区操作规则（新增）

```
当价格到达阻力区时：
1. 不急着卖，观察 30 分钟~1 小时
2. 如果放量突破阻力位 → 阻力变支撑，继续持有
3. 如果缩量受阻回落 → 减仓 1/3
4. 如果放量跌破阻力位（假突破）→ 减仓 2/3
5. 大单封涨停 → 全仓持有不动
```
