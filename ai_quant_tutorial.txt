1.4 确定核心指标：每个人不同，限免这是我自己关注的指标，你可以自
己根据自己的需求调整)

均线（MA）— 股价的体温计
把过去 N 天的收盘价加起来除以 N，就是 N 日均线。
  • MA5（5 日均线）：短期趋势，像这周的心情好不好
  • MA20（20 日均线）：中期趋势，像这个月的状态怎么样
核心用法：
  • 股价从下往上穿过 MA20 → 叫金叉，可能要涨（买入信号）
  • 股价从上往下跌破 MA20 → 叫死叉，可能要跌（卖出信号）
 ⚠ 想象 MA20 是一条河的水面。鱼（股价）从水下跳出来，说明它有力气（金叉）；鱼又沉回水
 下，说明没劲了（死叉）。

MACD — 动力是否还在
MACD 的计算很复杂，但你只需要知道一件事：
  • MACD > 0：上涨的动力还在，像油门还在踩着
  • MACD < 0：动力在衰减，像脚已经松开油门了
我们的策略中，MACD 通常配合均线一起看：价格站上 MA20 且 MACD > 0，才算有效的买
入信号。

成交量比（量比）— 今天是不是特别热闹
量比 = 今天的成交量 / 过去 5 天平均成交量
  • 量比 = 1.0：和平时差不多
  • 量比 = 2.0：今天的交易量是平时的 2 倍，特别热闹
  • 量比 = 0.5：今天很冷清
为什么重要？ 股价突破均线的时候，如果同时放量（量比 > 1.5），说明有大量资金在参与，
突破更可靠。如果缩量突破（量比 < 1），可能是假突破。

乖离率（Bias）— 跑得太远了没有
乖离率 = (当前股价 - MA20) / MA20 × 100%
  • 乖离率 = +8%：股价已经比均线高了 8%，跑得太远了，容易回调
  • 乖离率 = -8%：股价比均线低了 8%，跌得太多了，可能有反弹机会
  • 乖离率在 ±3% 之间：正常波动
核心作用：防追高。 即使股价站上 MA20，如果乖离率已经超过 5%，说明这趟车你上晚了，
等回调再说。

止损/止盈规则
市场           止盈                          止损                    最长持有天数

A股           跌破 MA20 卖出                  12%（A 股波动大）           30 天

美股           +20~25% 或跌破 MA20            8%（无涨跌停保护）            30 天


  ⚠ 回测告诉我们的真相：所有花哨的止盈策略在回测中都不如简单的涨到就卖、跌到就砍。简单粗
  暴，但有效。


资金管理
  •   单只股票最多投入总资金的 30%，不要把鸡蛋放在一个篮子里
  •   新手用总资金的 10% 试水，验证策略有效后再逐步加仓
  •   永远留有现金，市场暴跌时你才有子弹抄底


回测的三大陷阱

1. 过拟合
你在历史数据上调出了一套完美参数（年化 50%！），但一用到实盘就不灵了。
原因：参数恰好拟合了历史上的特殊行情，不具有普适性。


                           AI 辅助量化交易完全教程 | 第 8 页
  •   不要追求最优参数，选择中间偏好的参数
  •   在不同时间段上验证，2021 年有效的策略 2023 年还有效吗？

2. 幸存者偏差
你用的股票列表是现在还存在的公司。那些退市、破产的股票不在你的测试范围里，导致回测
结果偏乐观。
应对：意识到回测结果会比实际偏好 2~5 个百分点。

3. 滑点与流动性
回测假设你能以收盘价/开盘价精确成交，但实际上：
  • 小盘股买卖价差很大
  • 你下单的一瞬间，价格可能已经变了
应对：在回测中加入交易成本（已加了 0.1%），实盘预期收益打个 8 折。


模拟盘的重要性
在投入真金白银之前，至少用模拟盘跑两周：
  • A 股模拟盘：很多券商 App 提供
  • 美股模拟盘：Tiger Brokers 有模拟交易模式（Paper Trading）
模拟盘能帮你发现代码 bug、网络延迟、滑点等实盘中会遇到的问题。

B. Python 环境配置详细步骤

Windows 用户
  •   下载 Python：访问 python.org，下载最新版本（3.12 或更新），安装时务必勾选 Add
      Python to PATH
  •   验证安装：Win + R 输入 cmd 回车，然后输入 python --version
  •   安装依赖包：pip install akshare pandas numpy matplotlib yfinance
  •   国内加速：pip install akshare -i https://pypi.tuna.tsinghua.edu.cn/simple
  •   创建工作目录：在 D 盘创建文件夹，比如 D:\\stocks_ai\\

Mac 用户
  •   安装 Homebrew（如果没有的话）：/bin/bash -c 安装脚本
  •   安装 Python：brew install python
  •   安装依赖：pip3 install akshare pandas numpy matplotlib yfinance
