# A股主力行为分析框架

> 一套基于实盘验证的A股主力行为判别体系，从L2盘口到资金流分析的完整方法论。

## 📖 框架概览

本框架将A股主力分为**四种类型**，每种类型有独立的分析方法和操作策略：

| 类型 | 代表票 | 核心特征 | 分析方法 |
|------|--------|---------|---------|
| 游资出货型 | 鹭燕医药(002788) | 人数骗局，笔数多+每笔极小 | L2盘口⭐⭐⭐⭐⭐ |
| 庄家出货型 | 奥瑞德(600666) | 金额骗局，巨量托单对倒 | L2盘口⭐⭐⭐⭐ |
| 大基金锁仓型 | 华虹公司(688347) | 被动交易，盘口稀疏 | K线+资金流⭐⭐⭐⭐ |
| 资金流驱动型 | 中船特气(688146) | 基金申购推动，轻资产高ROE | 资金流⭐⭐⭐⭐⭐ |

## 🔑 核心认知

1. **主力只有一个目的：出货。** 不要用"洗筹""吸筹"来美化下跌。
2. **价格下跌到支撑位 = 出货成功的证据。** 不是买点。
3. **出货失败才是买点。** 主力想卖但卖不掉，价格自然反弹。
4. **基金申购赎回 = 油门，财报 = 涡轮增压。**
5. **笔数为零 = 假单。** 不管多大单子，不成交就是装饰品。
