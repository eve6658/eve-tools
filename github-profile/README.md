## 💡 核心观点

> **基金申购赎回 = 油门（决定方向）**
> **企业财报 = 涡轮增压（决定速度）**
> 有钱+财报好→涨得猛 | 有钱+财报普通→也涨 | 没钱→财报再好也涨不动

> **价格下跌到支撑位 ≠ 买点**
> = 出货成功的证据
> 出货失败才是买点（主力想卖但卖不掉）
