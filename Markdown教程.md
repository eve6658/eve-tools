# Markdown 完整教程

## 什么是 Markdown？

Markdown 是一种轻量级标记语言，用简单易读的纯文本格式编写文档，然后转换成有效的 XHTML（或 HTML）。它被广泛用于：
- 技术文档、博客、笔记
- GitHub README、Wiki
- 即时通讯中的富文本消息

---

## 1. 标题

用 `#` 号表示标题，`#` 数量对应标题级别（1-6级）。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

**效果：**
# 一级标题
## 二级标题
### 三级标题
#### 四级标题

---

## 2. 文本格式

| 语法 | 效果 | 说明 |
|------|------|------|
| `*斜体*` | *斜体* | 用星号或下划线包裹 |
| `_斜体_` | _斜体_ | 效果同上 |
| `**粗体**` | **粗体** | 双星号或双下划线 |
| `__粗体__` | __粗体__ | 效果同上 |
| `***粗斜体***` | ***粗斜体*** | 三者组合 |
| `~~删除线~~` | ~~删除线~~ | 双波浪线 |
| `` `行内代码' `` | `行内代码` | 反引号包裹 |

---

## 3. 列表

### 无序列表

用 `-`、`*` 或 `+` 开头：

```markdown
- 苹果
- 香蕉
- 橙子
```

**效果：**
- 苹果
- 香蕉
- 橙子

### 有序列表

用数字加点：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

**效果：**
1. 第一步
2. 第二步
3. 第三步

### 嵌套列表

```markdown
- 水果
  - 苹果
  - 香蕉
- 蔬菜
  - 白菜
  - 西红柿
```

**效果：**
- 水果
  - 苹果
  - 香蕉
- 蔬菜
  - 白菜
  - 西红柿

## 4. 链接与图片

### 链接

```markdown
[显示文本](https://example.com)
[带标题的链接](https://example.com "鼠标悬停显示")
```

**效果：**
[显示文本](https://example.com)

### 图片

```markdown
![图片描述](https://example.com/image.png)
```

---

## 5. 引用

用 `>` 开头：

```markdown
> 这是一段引用
> 可以多行
>
> > 还可以嵌套引用
```

**效果：**
> 这是一段引用
> 可以多行
>
> > 还可以嵌套引用

---

## 6. 代码

### 行内代码

用反引号包裹：`` `代码` ``

### 代码块

用三个反引号包裹，可指定语言：

\`\`\`python
def hello():
    print("Hello, World!")
\`\`\`

**效果：**
```python
def hello():
    print("Hello, World!")
```

---

## 7. 表格

用 `|` 分隔列，用 `-` 分隔表头和内容：

```markdown
| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25 | 北京 |
| 李四 | 30 | 上海 |
| 王五 | 28 | 广州 |
```

**效果：**

| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25 | 北京 |
| 李四 | 30 | 上海 |
| 王五 | 28 | 广州 |

### 对齐控制

```markdown
| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|-------:|
| left   | center | right |
```

| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|-------:|
| left   | center | right |

---

## 8. 分隔线

用三个或以上的 `-`、`*` 或 `_`：

```markdown
---
***
___
```

**效果：**

---
***
___

---

## 9. 任务列表

```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个任务
```

**效果：**
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个任务

---

## 10. 链接和脚注

### 自动链接

用尖括号包裹 URL 或邮箱：
```
<https://example.com>
<user@example.com>
```

### 脚注

```markdown
这里有一段话[^1]

[^1]: 这是脚注的内容
```

---

## 11. 高亮与特殊符号

### 高亮（部分平台支持）

```markdown
==高亮文字==
```

### 上标和下标（部分平台支持）

```markdown
x^2^（上标）
H~2~O（下标）
```

---

## 12. 转义字符

如果需要显示 Markdown 语法中的特殊符号，在前面加反斜杠：

```markdown
\*这不是斜体\*
\# 这不是标题
\[这不是链接\]
```
