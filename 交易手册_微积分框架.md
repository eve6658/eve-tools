# 第X章 资金流动的微积分框架

> 基于紫微斗数数学体系与盘口实盘验证，2026-04-30 整理
>
> **资金是水，遇山而停。** 山 = 价格墙 = 势能屏障。水积则溢，墙撤则涌。

---

## 一、核心定理

### 1.1 传统技术分析的致命缺陷

**禁止用历史股价推断当下趋势。**

均线、KDJ、MACD 等指标是"过去时空的资金流向"，不是下一个时空的。用过去的快照预测现在的运动，等于用照片预测物体的运动方向——丢失了动量信息。

同样，研报中引用"历史股价支撑/阻力位"来评价当下股价趋势，是错误的思维方式。价位本身没有意义，价位背后的**资金流动数学结构**才有意义。

### 1.2 微分与积分的角色分工

| 维度 | 微分 | 积分 |
|------|------|------|
| 本质 | 读取当下时空资金流动惯性的方向 | 用历史数据验证数学结果是否符合惯性 |
| 输入 | 实时逐笔成交、挂单变化、分时斜率 | 过往资金流动的数学结构（非股价） |
| 输出 | 瞬时动量方向 + 前一时空的特征延续 | 惯性是否自洽的判断 |
| 动作 | 自洽 → 执行；不自洽 → 观望 | |

**关键**：微分的结果自带"前一时空的特征"——惯性不会凭空消失，前一秒的方向会延续到下一秒。

### 1.3 判断逻辑

```
微分读方向 → 积分验真伪 → 吻合则动手
```

- 微分结果与积分趋势一致 → 惯性延续，趋势可信
- 微分结果偏离积分趋势 → 当下为异常值，可能是转折信号（外力介入）

类比物理：
- 微分 = 瞬时速度（带着前一刻的动量）
- 积分 = 总位移（历史轨迹的累积）
- 瞬时速度和总位移趋势方向一致 → 物体按惯性运动
- 瞬时速度突然偏离位移趋势 → 有外力介入（主力动作）

## 二、大/小盘股的根本差异

### 2.1 套牢盘规则

| 类型 | 套牢盘 | 积分回测逻辑 |
|------|--------|-------------|
| 大盘股（万亿级） | 存在，形成真实阻力 | 积分含历史价位结构（套牢盘、密集区） |
| 小盘股/科技类 | **不存在** | 积分只看资金流动本身的数学惯性，不看价位阻力 |

**核心纠正**：小盘股主力可以完全控盘，历史价位对当前价格没有约束力。用"历史支撑位/阻力位"分析小盘股是根本性错误。

### 2.2 积分回测周期

- 不同股票不同回测数据，20天到半年
- 取决于主力操盘周期、流通盘大小
- 可用**成交量加权的时间窗口**自适应：近期权重大、远期权重小，衰减到成交量贡献趋近于零时截断
- 每只股票自动算出自己的"有效记忆长度"

## 三、资金流动的数学特征（观测维度）

### 3.1 量价关系

- 下跌缩量 = 主力锁仓慢慢出；下跌放量 = 真抛压
- **成交额/跌幅** = 单位跌幅消耗的资金量，越大说明承接越强

### 3.2 逐笔成交分布

- 单笔手数的**均值和方差**：方差小 = 程序化操作；方差大 = 自然交易
- B/S 方向的**时间聚类**：连续 B 单扎堆 = 机构扫货；散落 B 单 = 散户抄底

### 3.3 委托挂单的幂律特征

- 正常市场挂单量服从幂律分布
- 某价位突然出现异常厚单（偏离分布 2σ 以上）= 人为干预信号
- 撤单率高 = 诱多/诱空

### 3.4 资金流的分形维度

- 分时成交量的 Hurst 指数：>0.5 趋势延续，<0.5 均值回归
- 粗略判断：连续 N 个 5 分钟 K 线成交量是递增还是随机

### 3.5 小单占比异动与主力伪装识别

- **吸筹特征**：卖盘小单 >80% + 买盘小单 <30% + 价格稳定 + 铁底未动
- **出货特征**：买盘小单 >80% + 卖盘小单 <30%（散户接盘）
- 初步阈值来自 2026-04-02 实盘数据（600666 奥瑞德、002788 鹭燕医药）

**主力伪装小单识别：**

主力也会用小单伪装成散户，不能光看小单占比。

| 特征 | 真散户 | 主力伪装 |
|------|--------|----------|
| 频率 | 随机、无规律（泊松分布） | 均匀、有节奏（程序化） |
| 单笔手数 | 参差不齐 | 异常一致（方差极小） |
| 时间间隔 | 随机到达 | 固定间隔（如每3秒一笔） |
| 价格 | 随意挂单 | 贴着同一价位反复出 |

**实盘案例（大基金伪装 - 压盘吸筹）：**
- 基底节奏：1秒1单1手，稳定输出（方差≈0）
- 扰动层：偶尔插入**几百手甚至上千手**的单子（数量级跳跃）
- **卖盘特征**：小单碎单（1-10手）→ 主力分拆卖单，控制下跌斜率，伪装散户卖出
- **买盘特征**：大单集中（100-1259手）→ 不是吸筹，是**价格墙**，阻止散户在低位买入
- **本质**：卖的是主力（碎单控速），买盘大单是**托价墙**（挡住散户的买路），散户被挤出去了
- **数学结构**：小卖单维持下降惯性（微分方向向下但加速度可控），大买单墙封锁散户买入空间
- **价格墙识别**：大单子集中出现在一个价位区间 → 这就是价格墙，资金流动的**势能**
- **势能→动能**：墙在 = 势能积累（价格锁死），墙薄 = 势能减弱（价格缓跌/缓涨），墙撤 = 动能释放（价格突变）
- **买盘墙打薄 → 股价下跌**：主力逐步撤掉买盘大单，支撑减弱，价格自然下滑——这是洗盘的核心手法，不是一次砸穿，而是慢慢抽水
- **卖盘墙打薄 → 股价上涨**：主力逐步撤掉卖盘大单，压力减弱，价格自然上行——同理，慢慢加水
- **两种墙的组合**：
  - 洗盘：小卖压价 + **大买墙封路**（封下方）→ 散户被压着，割肉后主力撤墙拉涨
  - 出货：小买拉价 + **大卖墙封顶**（封上方）→ 散户被堵着，追高后主力撤墙砸盘
  - 盘整：买墙+卖墙同时存在 → 价格被夹在中间，碎单在墙内波动
- **实盘验证**：002378 章源钨业 2026-04-30，卖墙5017万 vs 买墙2277万 → 卖墙是买墙2.2倍，天花板比地板厚，涨不动
- 真散户不可能稳定到1秒1单——人有思考、犹豫、网络延迟，间隔是随机的
- **2026-04-30 验证**：600666 奥瑞德，价格5.27-5.29窄幅锁定，同一时间段华虹(688347)、中船特气、中芯也出现同样模式 → 同一拨大基金并行运作

**判断公式：**
- 小单占比高 + 频率均匀 + 手数方差小 = **主力伪装**
- 小单占比高 + 频率随机 + 手数方差大 = **真散户**

### 3.6 股东人数的局限性与三角色轮转模型

**数据时效性：**
- **6字头（上交所）延后3个月** → 全部放弃，不做参考
- **0字头/3字头（深交所）部分10天更新** → 每10天跟踪，定位主力当前位置
- 实时判断主力行为 → 靠逐笔成交的小单占比，不靠股东人数

**三角色轮转模型：**

股东人数变化的本质不是"筹码集中度"，而是三个角色的轮转游戏：

1. **散户** — 第一棒。热度起来时先冲进来 → 人数增加
2. **小基金** — 散户攒够筹码后，小基金出概念进场，一人头算一户 → 人数急剧减少
3. **大股东** — 发现这么多人进来就开始丢货，一户变多户 → 人数又增加

三个角色来回窜动，不断循环。

**终局信号：**
- 当股东人数达到最近一年的峰值 = 所有该信的人都已进场
- 随后最后一脚油门（井喷）→ 结束
- "2万-20万是炒作活跃度甜区"适用于判断过程阶段，不适用于终局判断

## 四、主力建仓/出货的二阶导数判断

- **加速跌**（二阶导为负）：卖压在增大，趋势强化
- **减速跌**（二阶导为正）：卖盘衰竭，惯性在消退，可能见底
- **匀速跌**：主力控盘出货，温水煮青蛙
- 关键不是"跌了"，是**跌的加速度**

## 五、操盘手法模式库

### 5.1 温水煮青蛙（阴跌出货）

- 特征：小高开 → 匀速阴跌，不砸盘，慢慢磨
- 数学特征：分时斜率恒定负值，方差极小（程序化控盘）
- 散户感知：不觉得痛，不跑
- 案例：600666 奥瑞德 2026-04-30

### 5.2 两头剥皮

- 特征：反复拉砸，制造波动收割
- 案例：600666 奥瑞德 2026-04-16

### 5.3 单边出货

- 特征：一刀切式下跌，无反复
- 案例：002788 鹭燕医药

## 六、易经序卦与交易纪律（附录）

- **需卦**（水天需）：顺应惯性，等，定价权在手
- **讼卦**（天水讼）：主观预判，证明自己对 → 交出定价权 → 亏损
- **师卦**（地水师）：用纪律重建仓位管理
- **核心定理**：永远不要证明自己是对的 → 不证明 → 定价权在手
