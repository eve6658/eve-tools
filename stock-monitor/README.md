# Stock Monitor - 踩油门指标研究

## 目标
连续3个月每日抓取4只股票的盘口数据，从中找到主力踩油门的数学标志。

## 股票列表
| 代码 | 名称 | 股东人数变化 | 股价变化 |
|------|------|------------|---------|
| 002594 | 比亚迪 | 20万→73万 | 137→88 |
| 601899 | 紫金矿业 | 33万→125万 | 44→29 |
| 601345 | 工业妇联 | 31万→73万 | 83→47 |
| 300274 | 阳光电源 | 17万→31万 | 209→122 |

## 待找的踩油门指标
- 价格墙变化速率（卖墙撤单速度/买墙撤单速度）
- 大单净流入/流出突变点
- 小单占比异常波动
- 买卖盘力量失衡临界值
- 惯性启动前的盘口特征

## 数据采集
- 需要L2千档盘口数据（通过orderbook_receiver接收）
- 每日采集时间：9:25集合竞价 / 9:30开盘 / 11:30午盘 / 13:00午开 / 14:30尾盘 / 15:00收盘
